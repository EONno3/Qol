# 결과 정산 항목 사전 1차

이 문서는 미션 결과 보고서에서 비용, 보상, 손실, 미획득 항목으로 정리되는 정산 항목을 다룬다.

중요:

- 최종 확정본이 아니다.
- 수치 밸런스는 아직 확정하지 않는다.
- 정산 항목은 상태 태그, 장비 변화, 임플란트 변화를 대체하지 않는다.
- 정산 항목은 결과 보고서의 비용/보상/손실/미획득 정리에 쓰인다.
- 원인은 사건 로그와 변화 항목에서 읽히고, 정산 항목은 그 결과를 숫자나 처리 항목으로 보여준다.

## 1. 작성 기준

기본 필드:

```text
정산 항목명
분류
발생 조건
플레이어 판독 단서
결과 보고서 표시 예시
관련 변화/상태
오용 금지
```

표기 원칙:

- 정산 항목은 내부 원인을 설명하는 태그가 아니다.
- 손실과 추가 베네핏 미획득을 구분한다.
- 보험 적용은 보상이 아니라 손실 완화다.
- 수리/치료/재시술 비용은 원인 변화와 함께 읽혀야 한다.
- 플레이어에게 “정답”을 직접 설명하지 않는다. 정산은 사건 로그 해석을 보조한다.

## 2. 비용 항목

### 장비 정비비

- 분류: 비용.
- 발생 조건: 장비 변화가 정비, 부품 교체, 세척, 보정을 요구할 때.
- 플레이어 판독 단서: 외장 장비 손상이 다음 준비 비용으로 넘어온다.
- 결과 보고서 표시 예시: `장비 정비비: -120 크레딧.`
- 관련 변화/상태: 장비 변화 사전의 정비/손상/오염 계열.
- 오용 금지: 임플란트 정비비와 합산하지 않는다.

### 임플란트 정비비

- 분류: 비용.
- 발생 조건: 임플란트 변화가 전문 정비, 재동기화, 출력 보정, 부품 교체를 요구할 때.
- 플레이어 판독 단서: 몸 안 파츠 손상이 장비보다 무거운 복구 비용으로 넘어온다.
- 결과 보고서 표시 예시: `임플란트 정비비: -300 크레딧.`
- 관련 변화/상태: 기능 불안정, 임플란트 기능 저하, 동기화 오류, 파츠 과부하, 파츠 손상.
- 오용 금지: 임플란트 변화 자체를 정비비 항목으로 대체하지 않는다.

### 재시술비

- 분류: 고비용 복구.
- 발생 조건: 임플란트 파츠가 수리만으로 회복되지 않고 재시술을 요구할 때.
- 플레이어 판독 단서: 장비를 고치는 문제가 아니라 몸을 다시 열어야 하는 문제다.
- 결과 보고서 표시 예시: `재시술비: -900 크레딧.`
- 관련 변화/상태: 재시술 필요, 파츠 손상, 파츠 교체.
- 오용 금지: 일반 정비비와 같은 항목으로 뭉개지 않는다.

### 치료비

- 분류: 비용.
- 발생 조건: 용병 상태 태그나 부상/독성/신경계 후유증이 치료를 요구할 때.
- 플레이어 판독 단서: 임플란트 문제가 원인이었더라도 몸에 남은 피해는 따로 치료해야 한다.
- 결과 보고서 표시 예시: `치료비: -180 크레딧.`
- 관련 변화/상태: `#경미한_부상`, `#독성노출`, `#신경통`, `#회복중`.
- 오용 금지: 임플란트 정비비와 치료비를 자동 합산하지 않는다.

## 3. 손실 완화 항목

### 임플란트 보험 적용

- 분류: 손실 완화.
- 발생 조건: 보험 가입된 임플란트가 정비비, 재시술비, 파츠 교체비를 발생시킬 때.
- 플레이어 판독 단서: 신체 리스크가 사라지는 것이 아니라 비용 충격이 줄어든다.
- 결과 보고서 표시 예시: `임플란트 보험 적용: 재시술비 일부 차감.`
- 관련 변화/상태: 파츠 손상, 재시술 필요, 파츠 교체, 크롬 과열.
- 오용 금지: 보험 적용을 보상금처럼 표현하지 않는다.

### 장비 보험 적용

- 분류: 손실 완화.
- 발생 조건: 보험 또는 보증이 적용되는 장비가 손상/파손/분실됐을 때.
- 플레이어 판독 단서: 장비 손실 자체가 사라진 게 아니라 비용 일부가 줄어든다.
- 결과 보고서 표시 예시: `장비 보험 적용: 수리비 일부 차감.`
- 관련 변화/상태: 장비 변화 사전의 파손/손실/정비 계열.
- 오용 금지: 장비 회수 성공이나 추가 보상으로 표시하지 않는다.

## 4. 보상 항목

### 기본 보수

- 분류: 보상.
- 발생 조건: 미션 계약 조건을 충족했을 때.
- 플레이어 판독 단서: 의뢰 완료에 따른 기본 지급액이다.
- 결과 보고서 표시 예시: `기본 보수: +600 크레딧.`
- 관련 변화/상태: 미션 성공/부분 성공.
- 오용 금지: 추가 성과 보상과 섞지 않는다.

### 추가 성과 보상

- 분류: 보상.
- 발생 조건: 계약 외 추가 성과, 회수물 확보, 민간 피해 억제, 증거 확보 등 추가 조건을 달성했을 때.
- 플레이어 판독 단서: 미션을 끝낸 것 이상으로 좋은 결과를 냈을 때 붙는다.
- 결과 보고서 표시 예시: `추가 성과 보상: +180 크레딧.`
- 관련 변화/상태: 후속 단서 확보, 장비 회수, 증거 확보, 관계 개선.
- 오용 금지: 기본 보수를 쪼개서 추가 성과처럼 보이게 하지 않는다.

## 5. 손실 항목

### 장비 손실

- 분류: 손실.
- 발생 조건: 장비가 파손, 압수, 분실, 회수 불가 상태가 됐을 때.
- 플레이어 판독 단서: 실제 자산이 사라졌거나 복구 비용이 발생했다.
- 결과 보고서 표시 예시: `장비 손실: 소음기 파손 / 대체 필요.`
- 관련 변화/상태: 장비 변화 사전의 영구 손실, 파손, 회수 실패.
- 오용 금지: 얻지 못한 추가 장비를 손실로 쓰지 않는다.

### 평판 손실

- 분류: 손실.
- 발생 조건: 의뢰인, 지역 세력, 기업, 주민 네트워크의 신뢰가 떨어졌을 때.
- 플레이어 판독 단서: 당장 크레딧이 빠지지 않아도 다음 의뢰 조건이 나빠질 수 있다.
- 결과 보고서 표시 예시: `평판 손실: 지역 상점가 신뢰 하락.`
- 관련 변화/상태: 민간 피해, 오인 교전, 의뢰 조건 위반.
- 오용 금지: 내부 플래그를 그대로 노출하지 않는다.

## 6. 추가 베네핏 미획득

### 추가 보수 미획득

- 분류: 추가 베네핏 미획득.
- 발생 조건: 미션은 완료했지만 보너스 조건을 달성하지 못했을 때.
- 플레이어 판독 단서: 가진 것을 잃은 게 아니라 더 받을 수 있었던 것을 놓쳤다.
- 결과 보고서 표시 예시: `추가 보수 미획득: 침입자 장비 회수 실패.`
- 관련 변화/상태: 부분 성공, 회수 실패, 증거 부족.
- 오용 금지: 손실 항목으로 표시하지 않는다.

### 후속 단서 미획득

- 분류: 추가 베네핏 미획득.
- 발생 조건: 사건은 해결했지만 기록, 증거, 인물 단서, 파츠 출처 등을 확보하지 못했을 때.
- 플레이어 판독 단서: 다음 의뢰나 세력 정보로 이어질 가능성을 놓쳤다.
- 결과 보고서 표시 예시: `후속 단서 미획득: 회수 장비의 출처 확인 실패.`
- 관련 변화/상태: 기록 회수 실패, 장비 회수 실패, 침입자 도주.
- 오용 금지: 스토리 스포일러를 직접 쓰지 않는다.

### 회수물 미획득

- 분류: 추가 베네핏 미획득.
- 발생 조건: 현장 장비, 파츠, 기록 장치, 증거물을 회수할 수 있었지만 확보하지 못했을 때.
- 플레이어 판독 단서: 실제 보유 자산을 잃은 것이 아니라, 현장에서 챙길 수 있었던 자원을 놓쳤다.
- 결과 보고서 표시 예시: `회수물 미획득: 침입자 장비 회수 실패.`
- 관련 변화/상태: 장비 회수 실패, 침입자 도주, 현장 통제 실패.
- 오용 금지: 이미 보유하던 장비를 잃은 경우에는 장비 손실로 처리한다.

### 현장 통제 보너스 미획득

- 분류: 추가 베네핏 미획득.
- 발생 조건: 미션은 해결했지만 직접 제압, 오인 교전 방지, 민간 피해 억제 같은 현장 통제 조건을 달성하지 못했을 때.
- 플레이어 판독 단서: 목표는 끝냈지만 현장을 깔끔하게 정리하지 못해 추가 이득을 놓쳤다.
- 결과 보고서 표시 예시: `현장 통제 보너스 미획득: 직접 제압 실패.`
- 관련 변화/상태: 직접 제압 실패, 오인 교전, 민간 피해, 침입자 도주.
- 오용 금지: 미션 실패나 실제 손실로 표시하지 않는다.

3. 상태 태그별 치료비/휴식/스테이션 회복 규칙 연결.
