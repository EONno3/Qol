# 임플란트 변화 사전 1차

이 문서는 내장형 무기, 신체 파츠, 보조 장기, 신경계, 연결구에 생기는 변화를 정리한 1차 사전이다.

중요:

- 최종 확정본이 아니다.
- 수리비, 재시술비, 출격 제한 수치는 아직 확정하지 않는다.
- 임플란트 변화는 외장 장비 변화보다 복구 비용과 신체 리스크가 크다.
- 임플란트 변화는 용병 상태 태그와 분리한다.
- 임플란트 문제로 나타난 신체 반응은 결과 로그에서 묘사할 수 있다.
- 정산 항목은 신체 반응 묘사가 아니라 실제 원인인 임플란트 변화로 기록한다.

## 1. 작성 기준

임플란트 변화는 모든 고장 증상을 따로 태그화하지 않는다.

기본 원칙:

- 변화명은 플레이어가 다음 준비를 다르게 판단할 만큼만 나눈다.
- 같은 판단을 만드는 세부 증상은 하나의 변화로 묶는다.
- 부위별 차이는 변화명을 늘리지 않고 `대상`, `부위별 적용`, `결과 보고서 표시 예시`에서 처리한다.
- 예: 시야 번짐, 열상 잔상, 청각 잡음, 화학 필터 혼선은 모두 `감각 노이즈`의 부위별 사례로 둔다.
- 예: 입력 지연은 별도 변화가 아니라 `신경 지연`의 부위별 사례로 둔다.

기본 필드:

```text
변화명
대상
성격
대표 효과
부위별 적용 (선택)
플레이어 판독 단서
다음 준비 판단
주요 변화 관계
함께 생길 수 있는 상태 (선택)
복구/정비 조건
상태 태그/장비 변화와 구분
결과 보고서 표시 예시
오용 금지
```

표기 원칙:

- 파츠명과 변화명을 같이 표시한다.
- `결과 보고서 표시 예시`는 플레이어에게 보이는 문구 예시다.
- 표시 문구 자체를 태그명으로 오해하지 않는다.
- `다음 준비 판단`은 제작자/밸런싱용 판단 기준이다. 플레이어 UI에 그대로 노출하지 않는다.
- `함께 생길 수 있는 상태`는 자동 전이가 아니다. 결과 판정에서 별도 원인이 확인될 때만 같은 결과 보고서에 따로 기록한다.
- 주요 변화 관계에 명시된 `파츠별 고유 변화`는 실제 시스템에 존재하는 태그 식별자가 아닙니다. 이는 해당 슬롯에 장착된 파츠 계열의 실제 고유 변화(예: 그립 오차, 내장 무기 과열 등)로 치환되어 상태 전이가 작동함을 의미하는 기획 규칙입니다.
- 임플란트 변화는 상태 태그를 대체하지 않는다.
- 정비비, 재시술비, 보험 적용 같은 정산 항목은 `결과 정산 항목 사전 1차.md`에서 다룬다.

금지:

```text
#호흡기_자극
#필터오염을 용병 상태 태그로 표시
#신경통으로 임플란트 파손 대체
#낮은_Frame
```

## 2. 범용 변화

범용 변화는 구체 파츠 변화가 아직 정해지지 않았을 때만 쓴다.

가능하면 감각계, 신경계, 생체 보조계, 운동계, 내장형 무기, 제조/보안 계열 고유 변화를 우선한다.

### 기능 불안정

- 대상: 임플란트 파츠, 내장형 무기, 보조 장기.
- 성격: 경고 단계 변화.
- 대표 효과: 다음 사용 시 고유 변화나 돌발 노드 발생 가능.
- 플레이어 판독 단서: 아직 망가진 것은 아니지만, 몸 안 장치가 임무 중 흔들릴 수 있다.
- 다음 준비 판단: 해당 파츠가 핵심 조건인 미션 전 점검한다.
- 주요 변화 관계:
  - 정상 → 기능 불안정
  - 기능 불안정 → 파츠별 고유 변화
  - 기능 불안정 → 임플란트 기능 저하
  - 기능 불안정 → 정상
- 복구/정비 조건: 점검, 출력 보정, 접속부 조정.
- 상태 태그/장비 변화와 구분: 결과 로그에는 몸의 반응을 묘사할 수 있다. 다만 원인이 임플란트 작동 문제라면 정산 항목은 임플란트 변화다.
- 결과 보고서 표시 예시:
  - `손목 연결구: 기능 불안정 / 장비 연동 중 접속 신호 끊김.`
  - `호흡계 임플란트: 기능 불안정 / 독성 구역 진입 후 보조 호흡 끊김.`
  - `내장 와이어 발사기: 기능 불안정 / 탈출 시 사출 반응 지연.`
- 오용 금지: 파츠별 고유 변화가 명확할 때 이 항목으로 뭉개지 않는다.

### 파츠 과부하

- 대상: 신경계, 순환계, 하완/하퇴 파츠, 호흡계, 고출력 내장형 무기.
- 성격: 누적 부담 변화.
- 대표 효과: 다음 출격에서 기능 불안정, 파츠 손상, 부정적 상태 태그 발생 가능.
- 플레이어 판독 단서: 임플란트가 이번 임무를 버텼지만 몸과 파츠 양쪽에 무리가 남았다.
- 다음 준비 판단: 같은 파츠를 다시 혹사하는 미션 전 휴식/정비를 고려한다.
- 주요 변화 관계:
  - 정상 → 파츠 과부하
  - 파츠 과부하 → 기능 불안정
  - 파츠 과부하 → 파츠 손상
  - 파츠 과부하 → 정상
- 복구/정비 조건: 휴식, 출력 제한, 임플란트 조정, 정비.
- 상태 태그/장비 변화와 구분: 파츠 과부하는 임플란트 변화다. 신체 후유증이 따로 남으면 상태 태그로 추가 가능하다.
- 결과 보고서 표시 예시: `신경계 보조 칩셋: 파츠 과부하 / 연속 대응 후 신경 피드백 지연.`
- 오용 금지: 곧바로 `#신경통` 같은 상태 태그로 대체하지 않는다.

### 출력 스파이크

- 대상: 순환계, 신경계, 근섬유 보조, 내장형 무기.
- 성격: 출력 변동 변화.
- 대표 효과: 성능 판정의 최저값과 최고값 폭이 커진다. 같은 파츠가 대성공과 치명적 실패를 모두 만들 수 있다.
- 플레이어 판독 단서: 약해진 것이 아니라, 임무 중 출력이 들쭉날쭉 튄다.
- 다음 준비 판단: 안정성이 중요한 미션 전 출력 제한이나 안정화 모듈을 적용한다.
- 주요 변화 관계:
  - 정상 → 출력 스파이크
  - 출력 스파이크 → 파츠 과부하
  - 출력 스파이크 → 출력 조절 실패
  - 출력 스파이크 → 반사 루프 불안정
  - 출력 스파이크 → 정상
- 복구/정비 조건: 출력 제한, 안정화, 펌웨어 조정.
- 상태 태그/장비 변화와 구분: `#흥분`과 함께 발생할 수 있지만 같은 항목이 아니다.
- 결과 보고서 표시 예시:
  - `순환계 임플란트: 출력 스파이크 / 혈류 공급량이 임계치와 정지선 사이에서 종잡을 수 없이 요동침.`
  - `신경계 보조 칩셋: 출력 스파이크 / 신호 반응 속도가 극단적인 과부하와 무반응을 불규칙하게 오감.`
  - `근섬유 임플란트: 출력 스파이크 / 순간 동력이 한계 출력과 가동 중단 사이에서 불규칙하게 요동침.`
- 오용 금지: 단순 긍정 버프로 쓰지 않는다. 손/하완 제어 실패는 `출력 조절 실패`, 반응 과잉은 `반사 루프 불안정`, 열 누적은 `크롬 과열`로 구분한다.

### 동기화 오류

- 대상: 임플란트 파츠, 신경계 인터페이스, 외부 장비 접속부, 스마트 파츠.
- 성격: 연동/타이밍 변화.
- 대표 효과: 임플란트와 신경계, 임플란트끼리, 임플란트와 외부 장비 사이의 신호 타이밍이 어긋난다.
- 부위별 적용:
  - 감각계: 감각 노이즈, 오인식별, 표식 판독 지연으로 이어질 수 있다.
  - 하완/손: 그립 오차, 출력 조절 실패로 이어질 수 있다.
  - 내장형 무기: 내장 무기 걸림이나 발동 지연으로 이어질 수 있다.
  - 기업/군용 파츠: 펌웨어 충돌이나 인증 오류와 함께 발생할 수 있다.
- 플레이어 판독 단서: 파츠 하나가 망가진 것보다 여러 장치의 박자가 어긋난다.
- 다음 준비 판단: 시너지 장비, 스마트 장비, 정밀 조작 미션 전 재동기화한다.
- 주요 변화 관계:
  - 정상 → 동기화 오류
  - 동기화 오류 → 파츠별 고유 변화
  - 동기화 오류 → 임플란트 기능 저하
  - 동기화 오류 → 정상
- 복구/정비 조건: 재동기화, 펌웨어 점검, 접속부 조정.
- 상태 태그/장비 변화와 구분: 오인 경보는 사건 로그, 원인은 임플란트 동기화 오류일 수 있다.
- 결과 보고서 표시 예시:
  - `시각 보조 임플란트: 동기화 오류 / 표적 전환 중 화면-조준 보정 지연.`
  - `손목 연결구: 동기화 오류 / 스마트 장비 호출 한 박자 늦음.`
- 오용 금지: 모든 복합 실패를 동기화 오류로 뭉개지 않는다. 파츠별 원인이 명확하면 고유 변화를 우선한다.

### 임플란트 기능 저하

- 대상: 임플란트 파츠, 내장형 무기, 보조 장기.
- 성격: 범용 대체 변화.
- 대표 효과: 해당 임플란트가 읽히는 미션에서 신뢰도 하락 또는 추가 문제 발생 가능.
- 플레이어 판독 단서: 용병이 못한 게 아니라 몸 안 파츠가 제 역할을 덜 한다.
- 다음 준비 판단: 해당 파츠가 필수/권장 조건인 미션은 정비 전 피한다.
- 주요 변화 관계:
  - 기능 불안정 → 임플란트 기능 저하
  - 임플란트 기능 저하 → 파츠별 고유 변화
  - 임플란트 기능 저하 → 파츠 손상
  - 임플란트 기능 저하 → 정상
- 복구/정비 조건: 전문 정비, 부품 교체, 임플란트 조정.
- 상태 태그/장비 변화와 구분: 기능 저하는 임플란트 변화다. 신체 반응은 결과 로그에 따로 묘사한다.
- 결과 보고서 표시 예시: `호흡계 임플란트: 기능 저하 / 독성 환경에서 보조 호흡 유지 실패.`
- 오용 금지: `감각 노이즈`, `신경 지연`, `호흡 보조 출력 저하`처럼 더 구체적인 변화가 있으면 그쪽을 쓴다.

### 파츠 손상

- 대상: 시각, 청각, 하완, 하퇴/발, 신경계, 호흡계 등 임플란트 파츠.
- 성격: 강한 부정 임플란트 변화.
- 대표 효과: 수리 전까지 해당 파츠 기능 제한, 일부 미션 출격 제한 가능.
- 플레이어 판독 단서: 단순 정비가 아니라 특정 신체 기능이 미션 조건에서 빠질 수 있다.
- 다음 준비 판단: 파츠가 필수인 미션은 피하고, 수리/보험/대체 용병을 검토한다.
- 주요 변화 관계:
  - 파츠별 고유 변화 → 파츠 손상
  - 임플란트 기능 저하 → 파츠 손상
  - 파츠 손상 → 임플란트 기능 저하
  - 파츠 손상 → 정상
- 복구/정비 조건: 전문 수리, 부품 교체, 재시술.
- 상태 태그/장비 변화와 구분: 파츠 손상은 임플란트 변화다. 부상 상태와 함께 발생할 수 있지만 같은 항목이 아니다.
- 결과 보고서 표시 예시: `하완 임플란트: 파츠 손상 / 강제 개방 시 그립 보조 미작동.`
- 오용 금지: 파츠별 출격 불가 조건은 별도 판정한다.

### 재시술 필요

- 대상: 심각하게 손상된 임플란트 파츠, 기업/군용 파츠, 내장형 무기.
- 성격: 고비용 복구 변화.
- 대표 효과: 재시술 전까지 파츠 기능 회복 불가 또는 큰 제한.
- 플레이어 판독 단서: 장비 수리가 아니라 몸을 다시 열어야 하는 수준이다.
- 다음 준비 판단: 재시술 비용, 회복 시간, 보험 적용 여부를 먼저 본다.
- 주요 변화 관계:
  - 파츠 손상 → 재시술 필요
  - 재시술 필요 → 정상
- 복구/정비 조건: 새 파츠로 교체(결과 정산 항목 사전 연계) 또는 전문 클리닉 재시술.
- 상태 태그/장비 변화와 구분: 치료와 연결될 수 있지만 임플란트 복구 항목이다.
- 결과 보고서 표시 예시: `하퇴 임플란트: 재시술 필요 / 복귀 이동 중 하퇴 보조 미작동.`
- 오용 금지: 일반 정비 필요와 같은 비용감으로 다루지 않는다.

## 3. 감각계 변화

감각계 변화는 시각, 청각, 후각, 열감지, 감각 통합 파츠가 현장을 잘못 읽는 문제다.

세부 감각 오류를 별도 변화로 쪼개지 않는다.

### 감각 노이즈

- 대상: 시각 보조, 청각 보조, 후각/화학 감지, 열감지, 감각 통합 파츠.
- 성격: 감각계 판독 변화.
- 대표 효과: 감시, 추적, 검문, 제한 교전, 기록 회수 미션에서 현장 판독 오차.
- 부위별 적용:
  - 시각 보조: 표식, 얼굴, 조준선, 열상 판독 오차.
  - 청각 보조: 발소리, 경보음, 방향, 무전 잡음 분리 실패.
  - 후각/화학 감지: 독성, 향료, 약품, 화학 흔적 혼선.
  - 감각 통합 파츠: 서로 다른 감각 신호가 엇갈려 오인식별 발생.
- 플레이어 판독 단서: 감각이 없는 것이 아니라, 너무 많은 잡음이 섞여 현장을 잘못 읽을 수 있다.
- 다음 준비 판단: 감시/검문/제한 교전 미션 전 감각 보정이나 보조 장비를 준비한다.
- 주요 변화 관계:
  - 정상 → 감각 노이즈
  - 감각 노이즈 → 동기화 오류
  - 감각 노이즈 → 정상
- 복구/정비 조건: 감각 보정, 필터 조정, 재동기화.
- 상태 태그/장비 변화와 구분: `#예리함` 같은 상태 태그나 렌즈 오염 같은 장비 변화와 다르다.
- 결과 보고서 표시 예시:
  - `시각 보조 임플란트: 감각 노이즈 / 제한 교전 중 민간 신호를 위협으로 분류.`
  - `청각 보조 임플란트: 감각 노이즈 / 무전 잡음에 발소리 신호 묻힘.`
  - `후각 임플란트: 감각 노이즈 / 향료 잔류를 독성 흔적으로 판독.`
- 오용 금지: 단순 용병 판단 실수나 낮은 Pulse로 대체하지 않는다.

### 감각 기능 상실

- 대상: 시각 보조, 청각 보조, 후각/화학 감지, 열감지, 감각 통합 파츠.
- 성격: 감각계 강한 부정 변화.
- 대표 효과: 특정 감각 보조 기능이 임무 중 사실상 없는 것처럼 취급된다.
- 부위별 적용:
  - 시각 보조: 광학 보조 신호 끊김.
  - 청각 보조: 음향 보조 신호 끊김.
  - 후각/화학 감지: 화학 감지 신호 끊김.
- 플레이어 판독 단서: 잘못 읽는 정도가 아니라, 해당 감각 보조가 임무 조건에서 빠진다.
- 다음 준비 판단: 감각 의존 미션 전 정비하거나 대체 감지 장비/대체 용병을 준비한다.
- 주요 변화 관계:
  - 임플란트 기능 저하 → 감각 기능 상실
  - 파츠 손상 → 감각 기능 상실
  - 감각 기능 상실 → 정상 (전문 수리/교체 조치 전까지 자가 완화나 타 부정 상태로의 자연 전이 없음)
- 함께 생길 수 있는 상태: `#시야차단` (시각 보조 기능 상실 시), `#청각차단` (청각 보조 기능 상실 시)
- 복구/정비 조건: 새 파츠 교체 또는 전문 수리 (자가 치유나 자연 회복 없음).
- 상태 태그/장비 변화와 구분: 용병의 영구 장애나 정체성이 아니라 임플란트 보조 기능 상실이다.
- 결과 보고서 표시 예시:
  - `시각 보조 임플란트: 감각 기능 상실 / 광학 보조 신호 끊김.`
  - `청각 보조 임플란트: 감각 기능 상실 / 음향 보조 신호 끊김.`
- 오용 금지: `#맹인`, `#농인` 같은 상태명으로 대체하지 않는다. 상태 태그가 필요하면 별도 상태 사전에서 검토한다.

## 4. 신경계 변화

신경계 변화는 반응이 늦거나, 너무 튀거나, 통각 차단이 어긋나는 문제다.

### 신경 지연

- 대상: 신경계 칩셋, 하완/하퇴 제어부, 내장형 무기 인터페이스, 사이버덱 연결부.
- 성격: 신경계 임플란트 변화.
- 대표 효과: 반응, 회피, 정밀 조작, 플랜 B 발동, 해킹 입력이 늦어질 수 있다.
- 부위별 적용:
  - 하완/손: 정밀 입력, 락 해제, 장치 분리 지연.
  - 하퇴/발: 회피, 추격, 탈출 반응 지연.
  - 내장형 무기: 긴급 발동 지연.
  - 사이버덱 연결부: 해킹 입력 지연.
- 플레이어 판독 단서: 용병이 느린 게 아니라, 신호가 몸 끝까지 늦게 간다.
- 다음 준비 판단: 긴급 대응/정밀/전투 미션 전 신경계 조정을 한다.
- 주요 변화 관계:
  - 정상 → 신경 지연
  - 신경 지연 → 동기화 오류
  - 신경 지연 → 임플란트 기능 저하
  - 신경 지연 → 정상
- 복구/정비 조건: 신경계 보정, 칩셋 재동기화, 안정화.
- 상태 태그/장비 변화와 구분: `#신경통`은 신체 후유증, 신경 지연은 임플란트 반응 문제다.
- 결과 보고서 표시 예시:
  - `신경계 보조 칩셋: 신경 지연 / 기습 대응 시 반응 지연.`
  - `하완 신경 인터페이스: 신경 지연 / 락 해제 입력 한 박자 늦음.`
- 오용 금지: 낮은 Pulse나 낮은 Frame으로 대체하지 않는다.

### 반사 루프 불안정

- 대상: 신경계 반응 보조, 회피 보조, 근접전 보조 파츠.
- 성격: 과반응 변화.
- 대표 효과: 회피/근접 대응은 빨라질 수 있으나 제한 교전/정밀 작업에서 과잉 반응 발생 가능.
- 플레이어 판독 단서: 반응이 늦은 게 아니라 너무 빨리 튀어 현장을 흔들 수 있다.
- 다음 준비 판단: 제한 교전/검문 미션 전 반사 보조를 안정화한다.
- 주요 변화 관계:
  - 출력 스파이크 → 반사 루프 불안정
  - 반사 루프 불안정 → 파츠 과부하
  - 반사 루프 불안정 → 정상
- 복구/정비 조건: 반응 보정, 출력 제한.
- 상태 태그/장비 변화와 구분: `#흥분`은 정신 상태, 반사 루프 불안정은 신경계 임플란트 변화다.
- 결과 보고서 표시 예시: `신경계 반사 보조: 반사 루프 불안정 / 제압 상황에서 반응 과출력.`
- 오용 금지: 항상 좋은 반응 보너스로 쓰지 않는다.

### 통각 차단 오류

- 대상: 신경계 칩셋, 통각 차단 모듈, 군용 전투 보조.
- 성격: 신경계 보호 기능 변화.
- 대표 효과: 부상 감지 지연, 무리한 전투 지속, 부정적 상태 태그 발생 가능.
- 플레이어 판독 단서: 아픈 걸 못 느껴 버텼지만, 끝나고 대가가 드러날 수 있다.
- 다음 준비 판단: 전투/고위험 미션 전 통각 차단 설정을 점검한다.
- 주요 변화 관계:
  - 정상 → 통각 차단 오류
  - 통각 차단 오류 → 파츠 과부하
  - 통각 차단 오류 → 정상
- 함께 생길 수 있는 상태: `#경미한_부상`
- 복구/정비 조건: 신경계 설정 보정, 의료 점검.
- 상태 태그/장비 변화와 구분: 부상 상태는 결과로 따로 생길 수 있다. 통각 차단 오류는 임플란트 원인이다.
- 결과 보고서 표시 예시: `신경계 칩셋: 통각 차단 오류 / 교전 중 상처 확인 지연.`
- 오용 금지: 부상을 숨기는 장치로 남용하지 않는다. 결과 상태는 별도 공개한다.

## 5. 생체 보조계 변화

생체 보조계 변화는 호흡, 순환, 필터, 지혈처럼 몸의 유지 기능을 돕는 임플란트 문제다.

### 호흡 보조 출력 저하

- 대상: 호흡계 임플란트, 보조 폐, 환경 대응 임플란트, 산소 보조 파츠.
- 성격: 호흡/환경 대응 변화.
- 대표 효과: 수중, 독가스, 폐수 안개, 장시간 체류 미션에서 보조 호흡 신뢰도 하락.
- 플레이어 판독 단서: 특정 산소 압축 장치 문제가 아니라, 호흡 보조 계열이 전체적으로 힘을 덜 내는 상태다.
- 다음 준비 판단: 환경 미션 전 필터, 산소 보조, 외부 보호구를 함께 점검한다.
- 주요 변화 관계:
  - 정상 → 호흡 보조 출력 저하
  - 필터 포화 → 호흡 보조 출력 저하
  - 호흡 보조 출력 저하 → 임플란트 기능 저하
  - 호흡 보조 출력 저하 → 정상
- 함께 생길 수 있는 상태: `#독성노출`, `#피로누적`
- 복구/정비 조건: 출력 보정, 필터 교체, 호흡계 파츠 정비.
- 상태 태그/장비 변화와 구분: 호흡이 흔들린 로그는 묘사다. 정산 항목은 호흡 보조 출력 저하다.
- 결과 보고서 표시 예시: `호흡계 임플란트: 호흡 보조 출력 저하 / 수중 체류 중 보조 호흡 간헐 중단.`
- 오용 금지: 단순 피로와 자동 병합하지 않는다.

### 순환 보조 출력 저하

- 대상: 순환계 임플란트, 인공 심장, 혈관 보조 파츠, 나노혈액 보조.
- 성격: 순환/전투 지속 변화.
- 대표 효과: 장기 교전, 출혈 대응, 회복 보조 미션에서 신체 유지 보조 하락.
- 플레이어 판독 단서: 특정 펌프 한 개가 아니라, 몸 안에서 혈류와 회복을 밀어주는 계열이 약해진 상태다.
- 다음 준비 판단: 전투/장시간/의료 리스크 미션 전 순환계 안정화와 지혈 자원을 점검한다.
- 주요 변화 관계:
  - 정상 → 순환 보조 출력 저하
  - 지혈 나노머신 고갈 → 순환 보조 출력 저하
  - 순환 보조 출력 저하 → 출력 스파이크
  - 순환 보조 출력 저하 → 임플란트 기능 저하
  - 순환 보조 출력 저하 → 정상
- 함께 생길 수 있는 상태: `#출혈`, `#피로누적`
- 복구/정비 조건: 순환계 보정, 나노혈액 점검, 의료 안정화.
- 상태 태그/장비 변화와 구분: `#출혈`, `#피로누적`은 상태 태그다. 순환 보조 출력 저하는 순환계 임플란트 변화다.
- 결과 보고서 표시 예시: `순환계 임플란트: 순환 보조 출력 저하 / 장기 교전 중 혈류 보조 박동 밀림.`
- 오용 금지: 낮은 Frame이나 낮은 Cool로 대체하지 않는다.

### 필터 포화

- 대상: 호흡계, 후각/구강 보조 필터, 환경 대응 임플란트.
- 성격: 소모 부품 변화.
- 대표 효과: 환경 미션에서 보호 신뢰도 하락.
- 플레이어 판독 단서: 필터가 더 이상 받아낼 여유가 없어 다음 독성/먼지 환경에서 몸에 바로 부담이 갈 수 있다.
- 다음 준비 판단: 환경 미션 전 필터 교체를 우선한다.
- 주요 변화 관계:
  - 정상 → 필터 포화
  - 필터 포화 → 호흡 보조 출력 저하
  - 필터 포화 → 정상
- 복구/정비 조건: 필터 교체, 세척.
- 상태 태그/장비 변화와 구분: 용병이 독성에 노출됐으면 `#독성노출`, 필터가 한계에 찼으면 필터 포화.
- 결과 보고서 표시 예시: `호흡계 임플란트 필터: 포화 / 폐수 안개 통과 후 필터 한계 도달.`
- 오용 금지: `#필터오염`을 용병 상태 태그로 올리지 않는다.

### 지혈 나노머신 고갈

- 대상: 순환계 임플란트, 나노머신 혈액, 응급 지혈 모듈.
- 성격: 소모/의료 보조 변화.
- 대표 효과: 출혈/부상 노드에서 회복 실패나 치료비 증가 가능.
- 플레이어 판독 단서: 다음에 다치면 자동 지혈이 제때 작동하지 않을 수 있다.
- 다음 준비 판단: 전투/구출/고위험 미션 전 나노머신 보충.
- 주요 변화 관계:
  - 정상 → 지혈 나노머신 고갈
  - 지혈 나노머신 고갈 → 순환 보조 출력 저하
  - 지혈 나노머신 고갈 → 정상
- 함께 생길 수 있는 상태: `#출혈`
- 복구/정비 조건: 나노머신 보충, 의료 스테이션.
- 상태 태그/장비 변화와 구분: 실제 출혈은 상태 태그, 지혈 나노머신 고갈은 임플란트 준비 상태다.
- 결과 보고서 표시 예시: `순환계 임플란트: 지혈 나노머신 고갈 / 절상 대응 후 자동 지혈 재고 소진.`
- 오용 금지: 부상 자체를 이 항목으로 대체하지 않는다.

## 6. 운동계 변화

운동계 변화는 손, 하완, 하퇴/발, 관절부에서 생긴다. “힘이 약함”보다 “제어가 틀어짐”에 가깝다.

### 그립 오차

- 대상: 손 임플란트, 손가락 관절, 하완 제어부.
- 성격: 손/정밀 조작 변화.
- 대표 효과: 무기 파지, 락픽, 수리, 기록 장치 분리에서 실수 발생 가능.
- 플레이어 판독 단서: 손이 없는 게 아니라, 잡는 힘의 미세 조절이 어긋난다.
- 다음 준비 판단: 정밀/무기 조작 미션 전 손 파츠 보정.
- 주요 변화 관계:
  - 정상 → 그립 오차
  - 그립 오차 → 출력 조절 실패
  - 그립 오차 → 신경 지연
  - 그립 오차 → 정상
- 복구/정비 조건: 손가락 관절 보정, 하완 제어부 정비.
- 상태 태그/장비 변화와 구분: 정밀 도구 캘리브레이션 틀어짐은 장비 변화, 그립 오차는 손 임플란트 변화다.
- 결과 보고서 표시 예시: `손 임플란트: 그립 오차 / 소형 장치 조작 시 집게압 어긋남.`
- 오용 금지: 낮은 Wire나 낮은 Frame으로 대체하지 않는다.

### 출력 조절 실패

- 대상: 손, 하완, 상완, 인공 근섬유.
- 성격: 운동계 출력 변화.
- 대표 효과: 과도한 힘으로 장치 파손, 민간인 제압 과잉, 정밀 작업 실패 발생 가능.
- 플레이어 판독 단서: 힘이 부족한 게 아니라 너무 세거나 약하게 들어간다.
- 다음 준비 판단: 제한 교전/정밀 작업 전 출력 제한과 보정.
- 주요 변화 관계:
  - 그립 오차 → 출력 조절 실패
  - 출력 스파이크 → 출력 조절 실패
  - 출력 조절 실패 → 파츠 과부하
  - 출력 조절 실패 → 정상
- 복구/정비 조건: 출력 제한, 근섬유 보정, 하완 제어부 정비.
- 상태 태그/장비 변화와 구분: 용병 성격 문제가 아니라 운동계 임플란트 제어 문제다.
- 결과 보고서 표시 예시: `하완 임플란트: 출력 조절 실패 / 정밀 작업 중 파츠 과출력.`
- 오용 금지: 모든 Frame 관련 실패를 이 항목으로 처리하지 않는다.

### 균형 보정 실패

- 대상: 하퇴/발, 고관절, 반고리관 보조, 균형 제어 파츠.
- 성격: 이동/균형 변화.
- 대표 효과: 추격, 탈출, 고지대, 잠복 이동 미션에서 넘어짐/지연 발생 가능.
- 플레이어 판독 단서: 다리가 약한 게 아니라 자세를 잡는 보정이 늦거나 틀어진다.
- 다음 준비 판단: 추격/탈출/고저차 미션 전 균형 제어부 보정.
- 주요 변화 관계:
  - 정상 → 균형 보정 실패
  - 균형 보정 실패 → 감각 노이즈
  - 균형 보정 실패 → 파츠 손상
  - 균형 보정 실패 → 정상
- 복구/정비 조건: 하퇴 제어부 보정, 고관절 정렬, 감각 동기화.
- 상태 태그/장비 변화와 구분: 용병 피로와 별도다.
- 결과 보고서 표시 예시: `하퇴 임플란트: 균형 보정 실패 / 추격·탈출 중 자세 보정 지연.`
- 오용 금지: 지형 실패를 전부 하퇴 문제로 돌리지 않는다.

### 충격 흡수 저하

- 대상: 하퇴/발, 고관절, 장갑 관절, 낙하 완충 파츠.
- 성격: 이동/보호 변화.
- 대표 효과: 낙하, 돌진, 폭발 충격, 고중량 장비 미션에서 부상/파츠 손상 발생 가능.
- 플레이어 판독 단서: 이동은 가능하지만 충격을 받아내는 능력이 떨어진다.
- 다음 준비 판단: 고저차/폭발/중량 장비 미션 전 완충 파츠 점검.
- 주요 변화 관계:
  - 정상 → 충격 흡수 저하
  - 충격 흡수 저하 → 파츠 과부하
  - 충격 흡수 저하 → 정상
- 함께 생길 수 있는 상태: `#경미한_부상`
- 복구/정비 조건: 완충 관절 정비, 유압부 점검, 부품 교체.
- 상태 태그/장비 변화와 구분: 실제 부상은 상태 태그, 충격 흡수 저하는 임플란트 변화다.
- 결과 보고서 표시 예시: `하퇴 완충 파츠: 충격 흡수 저하 / 고저차 착지 후 충격 부담 증가.`
- 오용 금지: 모든 이동 실패를 이 항목으로 처리하지 않는다.

## 7. 내장형 무기 변화

내장형 무기 변화는 외장 무기보다 복구 비용이 크고, 몸에 붙어 있어 파츠 손상과 연결되기 쉽다.

### 내장 무기 과열

- 대상: 하완 투사체 런처, 구강 무기, 내장형 에너지 무기.
- 성격: 내장형 무기 변화.
- 대표 효과: 다음 교전에서 오발, 지연, 사용 제한 발생 가능.
- 플레이어 판독 단서: 플랜 B로 믿었던 내장 무기가 몸 안에서 뜨거운 리스크가 된다.
- 다음 준비 판단: 교전 미션 전 냉각 정비나 외장 예비 무기를 준비한다.
- 주요 변화 관계:
  - 정상 → 내장 무기 과열
  - 내장 무기 과열 → 크롬 과열
  - 내장 무기 과열 → 임플란트 기능 저하
  - 내장 무기 과열 → 정상
- 복구/정비 조건: 냉각, 정비, 부품 점검.
- 상태 태그/장비 변화와 구분: 열피로는 용병 신체 상태, 과열은 내장 무기 변화다.
- 결과 보고서 표시 예시: `하완 내장 투사체 런처: 과열 / 연속 발사 후 하완 열 누적.`
- 오용 금지: 외장 무기 과열과 비용/위험을 같은 층위로 다루지 않는다.

### 내장 무기 걸림

- 대상: 구강 독침, 하완 런처, 은닉 칼날, 와이어 발사기.
- 성격: 내장 무기 고유 퍼포먼스 저하.
- 대표 효과: 플랜 B나 조커 카드 발동 실패 가능.
- 플레이어 판독 단서: 위기 때 꺼내야 할 숨은 장비가 막힐 수 있다.
- 다음 준비 판단: 분해 정비하거나 외장 플랜 B를 추가한다.
- 주요 변화 관계:
  - 정상 → 내장 무기 걸림
  - 내장 무기 걸림 → 임플란트 기능 저하
  - 내장 무기 걸림 → 정상
- 복구/정비 조건: 분해 정비, 부품 교체.
- 상태 태그/장비 변화와 구분: 전투 실패 이유일 수 있지만 용병 상태는 아니다.
- 결과 보고서 표시 예시: `내장 와이어 발사기: 걸림 / 탈출 시 와이어 사출 지연.`
- 오용 금지: 외장 무기 탄걸림과 복구 비용을 동일시하지 않는다.

### 은닉 커버 노출

- 대상: 내장형 칼날, 하완 런처, 구강 무기, 접이식 포트.
- 성격: 내장형 무기 은닉 변화.
- 대표 효과: 검문/기업/잠입 미션에서 내장 무기 식별 가능.
- 플레이어 판독 단서: 무기는 작동해도 숨겨져 있다는 장점이 깨진다.
- 다음 준비 판단: 은닉 커버 수리, 검문 미션 회피, 외형 위장 조치.
- 주요 변화 관계:
  - 정상 → 은닉 커버 노출
  - 은닉 커버 노출 → Visibility 상승 가능
  - 은닉 커버 노출 → 정상
- 복구/정비 조건: 커버 교체, 외피 보수, 은닉 기믹 재정렬.
- 상태 태그/장비 변화와 구분: 외장 장비 `위장 손상`과 비슷하지만 몸 안 무기 은닉 문제다.
- 결과 보고서 표시 예시: `하완 내장 런처: 은닉 커버 노출 / 검문 스캔에 무기 윤곽 노출.`
- 오용 금지: 모든 내장형 무기를 자동 저가시성으로 보지 않는다.

## 8. 제조/보안 계열 변화

제조/보안 계열 변화는 파츠의 물리 기능뿐 아니라 제조 방식, 인증, 펌웨어, 신호 간섭을 다룬다.

### 바이오 거부 반응

- 대상: 바이오 임플란트, 유기체 보조 장기, 생체 접합부.
- 성격: 바이오 임플란트 고유 변화.
- 대표 효과: 장착/접합 단계의 근본 문제로 기능 이상, 치료비, 재시술 필요가 발생할 수 있다.
- 설정 기준:
  - 바이오 임플란트는 금속/기계 파츠보다 신체 접합과 회복에 강하다.
  - 독성 환경, 면역 교란, 과사용, 불량 클리닉 시술에 노출되면 몸이 파츠를 밀어내는 문제가 생긴다.
  - 이 문제는 기계 고장보다 의료 조치, 안정화, 재시술 판단에 더 강하게 연결된다.
- 플레이어 판독 단서: 파츠가 고장 난 것보다 몸이 파츠를 밀어내는 느낌으로 읽힌다.
- 다음 준비 판단: 의료 조치, 안정제, 바이오 파츠 조정이 필요하다.
- 주요 변화 관계:
  - 정상 → 바이오 거부 반응
  - 바이오 거부 반응 → 임플란트 기능 저하
  - 바이오 거부 반응 → 재시술 필요
  - 바이오 거부 반응 → 정상
- 함께 생길 수 있는 상태: `#신경통`, `#신경발열`, `#의체경련`
- 복구/정비 조건: 클리닉 면역 억제 조치, 생체 결합 점검 및 안정화 (필요 시 새 파츠 교체 연계).
- 상태 태그/장비 변화와 구분: 상태 태그는 결과로 함께 기록될 수 있지만, 원인은 바이오 임플란트 변화다.
- 결과 보고서 표시 예시: `바이오 호흡계 임플란트: 거부 반응 / 폐수 흡입 후 생체 접합부 붓기.`
- 오용 금지: 모든 바이오 파츠 손상을 거부 반응으로 처리하지 않는다.

### 회복 지연

- 대상: 바이오 임플란트, 자가 회복 파츠, 생체 접합부.
- 성격: 바이오 임플란트 회복 변화.
- 대표 효과: 수리비는 낮아도 회복 시간이 늘고 연속 출격에 불리.
- 플레이어 판독 단서: 거부 반응이 없어도 시술 후 접합부나 바이오 파츠가 아직 안정되지 않은 상태로 읽힌다.
- 다음 준비 판단: 연속 출격을 피하고 회복 시간을 확보한다.
- 주요 변화 관계:
  - 정상 → 회복 지연
  - 회복 지연 → 정상
- 함께 생길 수 있는 상태: `#회복중`
- 복구/정비 조건: 회복 시간, 의료 안정화, 식사/스테이션 회복.
- 상태 태그/장비 변화와 구분: `#회복중`은 용병 현재 상태, 회복 지연은 바이오 파츠 변화다.
- 결과 보고서 표시 예시: `바이오 하퇴 파츠: 회복 지연 / 야간 출격 후 접합부 회복 늦음.`
- 오용 금지: 모든 휴식 필요를 회복 지연으로 처리하지 않는다.

### 크롬 과열

- 대상: 크롬 임플란트, 군용 파츠, 고출력 내장형 무기.
- 성격: 크롬/군용 파츠 고유 변화.
- 대표 효과: 전투/장시간/고출력 미션에서 파츠 손상과 신체 부담 발생 가능.
- 플레이어 판독 단서: 성능은 강하지만 열과 비용으로 대가가 남는다.
- 다음 준비 판단: 냉각 정비, 출력 제한, 보험 적용 여부 확인.
- 주요 변화 관계:
  - 정상 → 크롬 과열
  - 크롬 과열 → 파츠 과부하
  - 크롬 과열 → 파츠 손상
  - 크롬 과열 → 정상
- 복구/정비 조건: 냉각 정비, 열 차폐, 부품 점검.
- 상태 태그/장비 변화와 구분: `#열피로`는 용병 신체 상태, 크롬 과열은 임플란트 변화다.
- 결과 보고서 표시 예시: `군용 하완 임플란트: 크롬 과열 / 고출력 제압 후 하완 열 누적.`
- 오용 금지: 모든 고성능 파츠를 자동으로 과열 처리하지 않는다.

### 신호 간섭

- 대상: 크롬 임플란트, 스마트 파츠, 신경계 칩셋, 내장형 무기.
- 성격: 크롬/전자계 변화.
- 대표 효과: 해킹, 감시망, 스마트 장비 시너지에서 오류 발생 가능.
- 플레이어 판독 단서: 몸 안 장치끼리 서로 방해해서 장비 조합이 흔들린다.
- 다음 준비 판단: 장비 조합 재검토, 신호 차폐, 시너지 점검.
- 주요 변화 관계:
  - 정상 → 신호 간섭
  - 신호 간섭 → 동기화 오류
  - 신호 간섭 → 펌웨어 충돌
  - 신호 간섭 → 정상
- 복구/정비 조건: 신호 차폐, 주파수 조정, 펌웨어 보정.
- 상태 태그/장비 변화와 구분: 외장 장비 `신호 누출`은 발각 문제, 신호 간섭은 임플란트 내부 간섭 문제다.
- 결과 보고서 표시 예시: `크롬 신경계 칩셋: 신호 간섭 / 스마트 조준과 신경 보정 신호 충돌.`
- 오용 금지: 모든 해킹 실패 원인으로 쓰지 않는다.

### 인증 오류

- 대상: 기업제 임플란트, 군용 파츠, 고급 스마트 파츠.
- 성격: 기업/군용 인증 변화.
- 대표 효과: 기업 보안, 검문, 파츠 시너지, 후속 단서에서 의심 발생 가능.
- 플레이어 판독 단서: 성능 문제가 아니라 “이 파츠가 누구 것인지”가 문제로 떠오른다.
- 다음 준비 판단: 기업/검문 미션 전 인증 갱신이나 위장 처리.
- 주요 변화 관계:
  - 정상 → 인증 오류
  - 인증 오류 → 펌웨어 충돌
  - 인증 오류 → Visibility 상승 가능
  - 인증 오류 → 정상
- 복구/정비 조건: 인증 갱신, 불법 패치, 기업 네트워크 우회.
- 상태 태그/장비 변화와 구분: 팩션 평판이나 수배와 별도다. 다만 결과로 노출/평판 변화가 생길 수 있다.
- 결과 보고서 표시 예시: `기업제 시각 보조 임플란트: 인증 오류 / 검문 기록 대조 중 소유자 코드 불일치.`
- 오용 금지: 이야기 플래그를 임플란트 변화처럼 직접 노출하지 않는다.

### 펌웨어 충돌

- 대상: 기업/군용 파츠, 스마트 임플란트, 신경계 칩셋, 내장형 무기.
- 성격: 소프트웨어/연동 변화.
- 대표 효과: 시너지 실패, 인증 지연, 보안망 의심, 조작 지연 발생 가능.
- 플레이어 판독 단서: 부품은 멀쩡해 보여도 내부 규칙이 충돌해 장비가 말을 안 듣는다.
- 다음 준비 판단: 펌웨어 점검, 인증 재등록, 기업 보안 미션 전 검증.
- 주요 변화 관계:
  - 정상 → 펌웨어 충돌
  - 펌웨어 충돌 → 동기화 오류
  - 펌웨어 충돌 → 임플란트 기능 저하
  - 펌웨어 충돌 → 정상
- 복구/정비 조건: 펌웨어 재설치, 인증 갱신, 스테이션 장비 분석.
- 상태 태그/장비 변화와 구분: 후속 단서나 해킹 흔적과 섞지 않는다.
- 결과 보고서 표시 예시: `기업제 시각 보조 임플란트: 펌웨어 충돌 / 보안 게이트 통과 중 인증 루프 지연.`
- 오용 금지: 이야기 플래그를 임플란트 변화처럼 노출하지 않는다.

## 9. 정산 항목 참조

임플란트 변화로 인해 비용이 발생할 수 있다.

정산 항목은 이 문서에서 직접 관리하지 않는다.

관련 정산 항목:

```text
임플란트 정비비
재시술비
임플란트 보험 적용
치료비
```

세부 규칙은 `결과 정산 항목 사전 1차.md`에서 다룬다.

## 10. 1차 보류 변화

아래 항목은 임플란트 변화로 바로 채택하지 않는다.

```text
#호흡기_자극
#신경통
#경미한_부상
#피로누적
외장 장비 오염
외장 장비 영구 손실
Visibility 상승
시야 번짐
열상 잔상
표적 오인
청각 잡음
방향감 상실
화학 필터 혼선
입력 지연
산소 압축 불안정
펌프 리듬 불안정
펌웨어 꼬임
```

보류 이유:

- 신체 증상 또는 상태 태그다.
- 외장 장비 변화다.
- `Visibility`는 장비 속성 변화로 장비 변화 사전에서 다룬다.
- 세부 감각/입력 증상은 더 큰 게임 판단 단위에 흡수한다.
- 특정 장치명에 묶인 변화명은 범용 계열명으로 대체한다.
- 임플란트 기능 변화와 그 결과 증상을 섞는다.
