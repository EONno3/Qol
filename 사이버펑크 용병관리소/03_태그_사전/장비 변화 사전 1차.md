# 장비 변화 사전 1차

이 문서는 외장 장비, 무기, 보호구, 도구, 소모품에 생기는 변화를 정리한 1차 사전이다.

중요:

- 이 문서는 최종 확정본이 아니다.
- 수리비, 교체비, 내구도 수치는 아직 확정하지 않는다.
- 장비 변화는 용병 상태 태그와 임플란트 변화와 분리한다.
- 이미 보유한 장비가 줄거나 망가지면 손실/정비 대상이다.
- 얻을 수 있었지만 얻지 못한 장비나 보너스는 `추가 베네핏 미획득`이다.

## 1. 작성 기준

장비 변화는 용병 몸 바깥의 장착물, 외장 무기, 보호구, 도구, 소모품에 생긴 결과다.

기본 필드:

```text
변화명
대상
성격
대표 효과
플레이어 판독 단서
다음 준비 판단
주요 변화 관계
복구/정비 조건
상태 태그/임플란트 변화와 구분
결과 보고서 표시 예시
오용 금지
```

표기 원칙:

- 범용 변화명은 일반 명사로 둔다.
- 장비명과 변화명을 같이 표시한다.
- `Visibility`는 태그가 아니라 장비 속성/수치다.
- `결과 보고서 표시 예시`는 플레이어에게 보이는 문구 예시다.
- 표시 문구 자체를 태그명으로 오해하지 않는다.

예:

```text
정밀 손가락 툴: 폐수 오염 / 세척 필요
소형 비살상 도구: 1회 소모
외장형 중화기: 경미한 파손 / 정비 필요
장비 노출도: 임시 수리 흔적으로 상승 가능
```

작성 방향:

- 장비 변화는 단순 정비 체크리스트가 아니다.
- 플레이어가 결과 보고서를 읽고 다음 미션 전 장비를 수리할지, 교체할지, 위장할지, 다른 장비를 들릴지 판단하게 만들어야 한다.
- 변화명은 짧게 유지하되, 결과 보고서 표시 예시와 판독 단서에서 장비별 개성을 드러낸다.

금지:

```text
#낮은_Visibility
#높은_Visibility
#Frame_저하
#피로누적
#경미한_부상
```

## 2. 오염/세척 계열

### 오염

- 대상: 외장 장비, 손 도구, 보호구, 외장 무기.
- 성격: 정비 필요 변화.
- 대표 효과: 다음 사용 시 신뢰도 저하, 정비비 발생 후보.
- 플레이어 판독 단서: 장비가 더러워진 정도가 아니라, 다음 현장에서 걸림/냄새/흔적/오작동으로 읽힐 수 있다.
- 다음 준비 판단: 환경/잠입/정밀 미션 전 세척하고, 거친 전투 미션은 위험을 감수할 수 있다.
- 주요 변화 관계:
  - 정상 → 오염
  - 오염 → 퍼포먼스 저하
  - 오염 → 정상
- 복구/정비 조건: 세척, 부품 점검, 스테이션 정비.
- 상태 태그/임플란트 변화와 구분: 용병 몸에 남은 독성은 `#독성노출`, 장비 표면이나 내부에 묻은 폐수/먼지는 오염.
- 결과 보고서 표시 예시: `정밀 손가락 툴: 폐수 오염. 세척 필요.`
- 오용 금지: 신체 후유증이나 환경 노출 상태로 뭉개지 않는다.

### 작동부 이물질

- 대상: 보호구 틈, 무기 작동부, 센서 포트, 도구 관절부.
- 성격: 경미한 정비 필요 변화.
- 대표 효과: 다음 1회 장시간 사용 시 오염이나 퍼포먼스 저하 위험.
- 플레이어 판독 단서: 움직이는 부위에 뭔가 끼어 있어 긴 사용 중 걸림이나 지연이 생길 수 있다.
- 다음 준비 판단: 짧은 미션은 감수 가능하지만, 정밀/장시간/잠입 미션 전에는 세척한다.
- 주요 변화 관계:
  - 정상 → 작동부 이물질
  - 작동부 이물질 → 오염
  - 작동부 이물질 → 정상
- 복구/정비 조건: 간이 세척, 점검.
- 상태 태그/임플란트 변화와 구분: 이물질은 장비 문제다.
- 결과 보고서 표시 예시: `센서 포트: 작동부 이물질 / 간이 세척 필요.`
- 오용 금지: 모든 오염을 같은 강도로 처리하지 않는다.

## 3. 소모/보급 계열

### 1회 소모

- 대상: 소형 비살상 도구, 일회용 드론, 약제 카트리지, 응급 키트.
- 성격: 소모품 변화.
- 대표 효과: 재보급 전까지 같은 장비 효과 사용 불가.
- 플레이어 판독 단서: 장비가 고장 난 것은 아니지만 같은 선택지를 다시 쓸 수 없다.
- 다음 준비 판단: 다음 미션에서 같은 대응이 필요하면 재보급하거나 대체 장비를 챙긴다.
- 주요 변화 관계:
  - 보유 → 1회 소모
  - 1회 소모 → 재보급 완료
- 복구/정비 조건: 재구매, 충전, 보급.
- 상태 태그/임플란트 변화와 구분: 도구 소모는 용병 상태가 아니다.
- 결과 보고서 표시 예시: `소형 비살상 도구: 1회 소모.`
- 오용 금지: 추가 베네핏 미획득과 혼동하지 않는다. 이미 가진 자원이 줄었으면 손실/소모다.

### 재보급 필요

- 대상: 탄약, 소모성 도구, 약제, 드론 배터리.
- 성격: 보급 필요 변화.
- 대표 효과: 재보급 전까지 일부 장비 효과 제한.
- 플레이어 판독 단서: 장비 성능보다 준비 상태가 빈칸으로 남아 있다.
- 다음 준비 판단: 보급 비용을 내거나, 해당 소모품이 필요 없는 미션을 고른다.
- 주요 변화 관계:
  - 1회 소모 → 재보급 필요
  - 재보급 필요 → 정상
- 복구/정비 조건: 보급 비용 지불, 스테이션 보급.
- 상태 태그/임플란트 변화와 구분: 장비 보급 문제다.
- 결과 보고서 표시 예시: `응급 키트: 재보급 필요.`
- 오용 금지: 정비 필요와 섞지 않는다. 재보급은 소모품 회복이다.

## 4. 충전/전력 계열

### 전력 부족

- 대상: 방어막, 전자 장비, 드론, 에너지 무기.
- 성격: 재충전 필요 변화.
- 대표 효과: 다음 출격 전 충전하지 않으면 일부 기능 제한.
- 플레이어 판독 단서: 장비가 망가진 것은 아니지만 긴 임무에서 중간에 꺼질 수 있다.
- 다음 준비 판단: 전력 소모가 큰 미션 전 충전하거나 보조 배터리를 준비한다.
- 주요 변화 관계:
  - 정상 → 전력 부족
  - 전력 부족 → 방전
  - 전력 부족 → 정상
- 복구/정비 조건: 충전, 배터리 교체, 스테이션 전력 사용.
- 상태 태그/임플란트 변화와 구분: 용병 피로가 아니라 장비 전력 문제다.
- 결과 보고서 표시 예시: `보조 방어막: 전력 부족 / 재충전 필요.`
- 오용 금지: `#피로누적` 같은 상태 태그로 대체하지 않는다.

### 방전

- 대상: 방어막, 전자 장비, 드론, 에너지 무기.
- 성격: 기능 제한 변화.
- 대표 효과: 재충전 전까지 장비 기능 사용 불가.
- 플레이어 판독 단서: 다음 미션에서 해당 장비를 없는 것처럼 봐야 한다.
- 다음 준비 판단: 충전 전 재출격 금지, 대체 장비 준비.
- 주요 변화 관계:
  - 전력 부족 → 방전
  - 방전 → 전력 부족
  - 방전 → 정상
- 복구/정비 조건: 충전, 배터리 교체.
- 상태 태그/임플란트 변화와 구분: 장비 작동 상태다.
- 결과 보고서 표시 예시: `정찰 드론: 방전 / 다음 출격 전 충전 필요.`
- 오용 금지: 파손과 혼동하지 않는다.

## 5. 내구/파손 계열

### 내구 저하

- 대상: 무기, 보호구, 드론, 외장 장비.
- 성격: 누적 정비 변화.
- 대표 효과: 반복 사용 시 경미한 파손 또는 퍼포먼스 저하 위험.
- 플레이어 판독 단서: 당장 고장은 아니지만 같은 장비를 계속 굴리면 다음 결과 보고서에서 더 큰 손실로 커질 수 있다.
- 다음 준비 판단: 저위험 미션은 감수 가능, 고위험/장시간 미션 전 정비 권장.
- 주요 변화 관계:
  - 정상 → 내구 저하
  - 내구 저하 → 경미한 파손
  - 내구 저하 → 정상
- 복구/정비 조건: 정비, 부품 교체.
- 상태 태그/임플란트 변화와 구분: 용병 피로와 별도다.
- 결과 보고서 표시 예시: `외장형 중화기: 내구 저하 / 정비 권장.`
- 오용 금지: 단일 미션의 직접 파손과 장기 내구도 변화를 섞지 않는다.

### 경미한 파손

- 대상: 외장 장비, 무기, 보호구, 드론.
- 성격: 수리 필요 변화.
- 대표 효과: 다음 사용 시 일부 기능 불안정, 수리비 발생.
- 플레이어 판독 단서: 장비는 아직 쓸 수 있지만 소리, 흔들림, 수리 흔적 때문에 성능과 `Visibility`가 동시에 흔들릴 수 있다.
- 다음 준비 판단: 잠입/기업/정밀 미션 전 수리, 거친 전투 미션은 위험 감수 가능.
- 주요 변화 관계:
  - 정상 → 경미한 파손
  - 내구 저하 → 경미한 파손
  - 경미한 파손 → 파손
  - 경미한 파손 → 정상
- 복구/정비 조건: 수리, 부품 교체.
- 상태 태그/임플란트 변화와 구분: 용병 부상과 별도다.
- 결과 보고서 표시 예시: `외장형 방어판: 경미한 파손 / 수리비 발생.`
- 오용 금지: 장비 파손을 `#경미한_부상`으로 표시하지 않는다.

### 파손

- 대상: 무기, 보호구, 드론, 외장 장비.
- 성격: 강한 부정 장비 변화.
- 대표 효과: 수리 전까지 장비 기능 제한 또는 사용 불가.
- 플레이어 판독 단서: 다음 미션 계획에서 해당 장비의 역할을 빼거나 대체해야 한다.
- 다음 준비 판단: 수리, 교체, 보험 처리, 대체 장비 장착 중 하나를 선택한다.
- 주요 변화 관계:
  - 경미한 파손 → 파손
  - 퍼포먼스 저하 → 파손
  - 파손 → 경미한 파손
  - 파손 → 정상
- 복구/정비 조건: 전문 수리, 부품 교체, 보험 처리.
- 상태 태그/임플란트 변화와 구분: 장비 문제다.
- 결과 보고서 표시 예시: `정찰 드론: 파손 / 수리 필요.`
- 오용 금지: 임플란트 파츠 손상과 같은 표로 뭉개지 않는다.

### 영구 손실

- 대상: 회수 실패 장비, 파괴된 소모품, 완전 손상 장비.
- 성격: 손실.
- 대표 효과: 재구매 필요.
- 플레이어 판독 단서: 이미 가진 자산이 사라졌으므로 다음 미션의 선택지가 줄어든다.
- 다음 준비 판단: 재구매, 보험 청구, 대체 장비 운용.
- 주요 변화 관계:
  - 파손 → 영구 손실
  - 영구 손실 → 재구매 완료
- 복구/정비 조건: 새 장비 구매, 보험 보상.
- 상태 태그/임플란트 변화와 구분: 장비가 사라진 것이지 용병 상태가 아니다.
- 결과 보고서 표시 예시: `정찰 드론: 회수 실패 / 영구 손실.`
- 오용 금지: 추가 베네핏 미획득과 혼동하지 않는다.

## 6. 기능/조작 계열

### 퍼포먼스 저하

- 대상: 무기, 전자 장비, 도구, 보호구.
- 성격: 부정 장비 변화.
- 대표 효과: 해당 장비가 읽히는 미션에서 신뢰도 하락.
- 플레이어 판독 단서: 장비가 작동은 하지만 “결정적인 순간에 삐끗할 수 있다”고 읽힌다.
- 다음 준비 판단: 해당 장비가 핵심 조건인 미션 전 정비한다.
- 주요 변화 관계:
  - 기능 불안정 → 퍼포먼스 저하
  - 퍼포먼스 저하 → 기능 불안정
  - 퍼포먼스 저하 → 정상
  - 퍼포먼스 저하 → 파손
- 복구/정비 조건: 정비, 부품 교체.
- 상태 태그/임플란트 변화와 구분: 장비 작동 문제다.
- 결과 보고서 표시 예시: `스마트 락픽: 퍼포먼스 저하 / 정비 필요.`
- 오용 금지: 용병의 기술 부족이나 스탯 부족으로 쓰지 않는다.
- 사용 제한: 구체 변화가 없을 때만 쓰는 상위 변화다. 가능하면 `탄걸림`, `렌즈 오염`, `신호 누출`, `밀폐 불량`처럼 장비별 고유 변화로 먼저 표현한다.

### 캘리브레이션 틀어짐

- 대상: 정밀 도구, 외장 무기, 원격 장비, 드론 조작 장비.
- 성격: 경고 단계 변화.
- 대표 효과: 정밀 작업이나 긴급 사용 시 돌발 노드 위험.
- 플레이어 판독 단서: 조준, 잠금 해제, 기록 추출, 드론 조작 같은 정밀 입력에서 오차가 생길 수 있다.
- 다음 준비 판단: 정밀/해킹/지원 미션 전 캘리브레이션을 맞춘다.
- 주요 변화 관계:
  - 정상 → 캘리브레이션 틀어짐
  - 캘리브레이션 틀어짐 → 퍼포먼스 저하
  - 캘리브레이션 틀어짐 → 정상
- 복구/정비 조건: 조정, 캘리브레이션, 정비.
- 상태 태그/임플란트 변화와 구분: 결과 로그에는 조작이 흔들린 장면을 쓸 수 있다. 다만 원인이 도구 보정 문제라면 정산 항목은 장비의 캘리브레이션 틀어짐이다.
- 결과 보고서 표시 예시: `정밀 툴: 캘리브레이션 틀어짐 / 재조정 필요.`
- 오용 금지: `Wire 부족` 같은 스탯 판정과 섞지 않는다.

## 7. 장비 유형별 고유 변화

### 무기 반동 증가

- 대상: 외장형 총기, 중화기, 동력 근접 무기.
- 성격: 무기 고유 변화.
- 대표 효과: Frame 요구가 높은 무기에서 명중/제어 위험 증가.
- 플레이어 판독 단서: 용병 스탯이 낮으면 장비가 용병을 끌고 다니는 상황이 된다.
- 다음 준비 판단: 반동 완화 부품 장착, 더 가벼운 무기 교체, Frame 높은 용병 배치.
- 주요 변화 관계:
  - 정상 → 무기 반동 증가
  - 무기 반동 증가 → 내구 저하
  - 무기 반동 증가 → 정상
- 복구/정비 조건: 반동 완화 부품, 정비, 장비 교체.
- 상태 태그/임플란트 변화와 구분: Frame 부족 자체가 아니라 장비 상태/개조 문제다.
- 결과 보고서 표시 예시: `외장형 중화기: 반동 증가 / 제어 부담 상승.`
- 오용 금지: `#낮은_Frame` 태그로 만들지 않는다.

### 발포음 증가

- 대상: 외장형 총기, 폭발물, 에너지 무기.
- 성격: 무기 고유 변화 / Visibility 상승 원인.
- 대표 효과: 잠입/검문/기업 미션에서 발각 위험 증가.
- 플레이어 판독 단서: 전투력은 유지돼도 “조용한 해결”이 어려워진다.
- 다음 준비 판단: 소음 억제 모듈 장착, 은닉형 무기 교체, 발각 감수 미션에만 사용.
- 주요 변화 관계:
  - 정상 → 발포음 증가
  - 발포음 증가 → Visibility 상승
  - 발포음 증가 → 정상
- 복구/정비 조건: 소음 억제 장치 정비, 총열/에너지 방출부 점검.
- 상태 태그/임플란트 변화와 구분: 용병 성향이 아니라 무기 흔적이다.
- 결과 보고서 표시 예시: `소음 억제 모듈: 손상 / 발포음 증가.`
- 오용 금지: 단순 전투 실패 원인으로만 쓰지 않는다.

### 탄걸림

- 대상: 실탄 화기, 투사체 무기.
- 성격: 무기 고유 퍼포먼스 저하.
- 대표 효과: 교전 중 조커 카드 실패 또는 지연 위험.
- 플레이어 판독 단서: 다음 교전에서 “한 번 막히면 위험한 장비”로 읽힌다.
- 다음 준비 판단: 분해 정비, 탄약 교체, 예비 무기 장착.
- 주요 변화 관계:
  - 정상 → 탄걸림
  - 탄걸림 → 퍼포먼스 저하
  - 탄걸림 → 정상
- 복구/정비 조건: 분해 정비, 탄약 점검.
- 상태 태그/임플란트 변화와 구분: 용병 반응 문제가 아니라 무기 작동 문제다.
- 결과 보고서 표시 예시: `실탄 화기: 탄걸림 흔적 / 분해 정비 필요.`
- 오용 금지: 모든 무기 퍼포먼스 저하를 탄걸림으로 처리하지 않는다.

### 렌즈 오염

- 대상: 드론, 카메라, 조준경, 센서 장비.
- 성격: 센서 고유 변화.
- 대표 효과: 감시/추적/기록 회수 미션에서 오인식별 또는 누락 위험.
- 플레이어 판독 단서: “봤지만 제대로 못 봤다”는 결과가 나올 수 있다.
- 다음 준비 판단: 렌즈 세척, 센서 교체, 보조 감지 장비 준비.
- 주요 변화 관계:
  - 정상 → 렌즈 오염
  - 렌즈 오염 → 퍼포먼스 저하
  - 렌즈 오염 → 정상
- 복구/정비 조건: 렌즈 세척, 센서 점검.
- 상태 태그/임플란트 변화와 구분: 용병 감각 상태가 아니라 장비 시야 문제다.
- 결과 보고서 표시 예시: `정찰 드론 렌즈: 오염 / 얼굴 기록 일부 흐림.`
- 오용 금지: 시각 임플란트 문제와 섞지 않는다.

### 신호 누출

- 대상: 드론, 원격 장비, 스마트 장비, 추적기.
- 성격: 전자 흔적 변화 / Visibility 상승 원인.
- 대표 효과: 보안 드론, 기업망, 해커 감시에 포착될 위험 증가.
- 플레이어 판독 단서: 물리적으로 숨겨도 전자적으로 들킬 수 있다.
- 다음 준비 판단: 신호 차폐, 펌웨어 점검, 원격 장비 사용 제한.
- 주요 변화 관계:
  - 정상 → 신호 누출
  - 신호 누출 → Visibility 상승
  - 신호 누출 → 정상
- 복구/정비 조건: 신호 차폐, 펌웨어 점검, 전자부 정비.
- 상태 태그/임플란트 변화와 구분: 장비 전자 흔적이다.
- 결과 보고서 표시 예시: `정찰 드론: 신호 누출 / 보안망 감지 위험.`
- 오용 금지: 이야기 단서나 후속 플래그와 같은 항목으로 쓰지 않는다.

### 밀폐 불량

- 대상: 방독 마스크, 보호복, 환경 보호구, 케이스.
- 성격: 보호구 고유 변화.
- 대표 효과: 환경 미션에서 보호 실패와 상태 태그 발생 위험.
- 플레이어 판독 단서: 보호구가 있어도 환경 위험을 완전히 막지 못한다.
- 다음 준비 판단: 실링 교체, 보호구 수리, 더 높은 등급 보호구 준비.
- 주요 변화 관계:
  - 정상 → 밀폐 불량
  - 밀폐 불량 → 퍼포먼스 저하
  - 밀폐 불량 → 정상
- 복구/정비 조건: 실링 교체, 점검, 보호구 수리.
- 상태 태그/임플란트 변화와 구분: 보호구 문제와 `#독성노출`은 함께 발생할 수 있지만 같은 항목이 아니다.
- 결과 보고서 표시 예시: `방독 보호구: 밀폐 불량 / 독성 노출 위험 증가.`
- 오용 금지: 호흡계 임플란트 기능 저하와 섞지 않는다.

### 위장 손상

- 대상: 위장복, 외장 커버, 무광 코팅, 장비 도색.
- 성격: Visibility 상승 원인.
- 대표 효과: 잠입/검문/기업 미션에서 장비 식별 위험 증가.
- 플레이어 판독 단서: 장비 성능은 남아도 “봐서는 안 될 것이 보인다.”
- 다음 준비 판단: 도색 보수, 외장 커버 교체, 은닉 장비 선택.
- 주요 변화 관계:
  - Visibility 감소 → 위장 손상
  - 위장 손상 → Visibility 상승
  - 위장 손상 → 노출 억제 유지
- 복구/정비 조건: 도색 보수, 코팅 재처리, 커버 교체.
- 상태 태그/임플란트 변화와 구분: 용병 은밀함 상태가 아니라 장비 외관 문제다.
- 결과 보고서 표시 예시: `외장 커버: 위장 손상 / 보안 드론 식별 위험 상승.`
- 오용 금지: `#위장손상` 같은 용병 상태 태그로 만들지 않는다.

## 8. Visibility 속성 변화

`Visibility`는 눈에 보이는 정도만 뜻하지 않는다.

검문, 잠입, 기업 보안, 드론 감시에서 장비가 들킬 확률 전체를 뜻하는 장비 속성이다.

따라서 시각 노출, 발포음, 작동음, 열 배출, 전자 신호, 화학 냄새, 군용/기업 식별 마킹을 모두 포함할 수 있다.

주의:

- `Visibility` 자체는 태그가 아니라 장비 속성/수치다.
- `Visibility`를 올리거나 낮추는 장비 특징, 개조, 도색, 소음 억제 옵션은 태그나 장비 옵션으로 존재할 수 있다.
- 결과 보고서에서는 `Visibility +n`만 쓰기보다 어떤 흔적 때문에 들켰거나 안정됐는지 사건 로그와 장비 변화로 보여준다.

### Visibility 상승

- 대상: 외장형 무기, 장비 조합, 임시 수리 흔적, 노출된 보호구.
- 성격: 장비 속성 변화.
- 대표 효과: 검문/잠입/기업/보안 미션에서 위험 증가 후보.
- 상승 원인 후보:
  - 화려한 장식.
  - 눈에 띄는 도색.
  - 거대한 실루엣.
  - 노출된 군용/기업 마킹.
  - 큰 발포음.
  - 거친 작동음.
  - 고열 배출.
  - 전자 신호 누출.
  - 화학 냄새나 오존 냄새.
  - 임시 수리 흔적.
- 주요 변화 관계:
  - 기본 Visibility → 일시 Visibility 상승
  - 일시 Visibility 상승 → 기본 Visibility
- 복구/정비 조건: 위장 처리, 장비 교체, 외장 커버 수리.
- 상태 태그/임플란트 변화와 구분: `Visibility`는 태그가 아니라 장비 속성/수치다.
- 결과 보고서 표시 예시: `장비 노출도: 임시 수리 흔적과 작동음으로 상승 가능.`
- 오용 금지: `#낮은_Visibility`, `#높은_Visibility` 같은 태그를 만들지 않는다.

### 노출 억제 유지

- 대상: 위장 장비, 은닉형 무기, 외장 커버.
- 성격: 장비 속성 정상 유지.
- 대표 효과: 보안 드론/검문 위협 점수 상승 없음.
- 안정 원인 후보:
  - 위장 도색.
  - 무광 코팅.
  - 소음 억제.
  - 열 차폐.
  - 신호 차폐.
  - 경량화.
  - 소형화.
  - 탈착식 외장 커버.
  - 군용/기업 마킹 제거.
  - 냄새 억제 처리.
- 주요 변화 관계:
  - Visibility 상승 → 노출 억제 유지
  - 노출 억제 유지 → Visibility 상승
- 복구/정비 조건: 위장 유지, 장비 은닉 상태 유지.
- 상태 태그/임플란트 변화와 구분: 용병 상태가 아니라 장비 노출 속성이다.
- 결과 보고서 표시 예시: `장비 노출 관련 기록: 위장 도색과 소음 억제 상태 유지.`
- 오용 금지: 긍정 상태 태그로 만들지 않는다.

### Visibility 감소

- 대상: 외장형 무기, 보호구, 드론, 도구, 장비 조합.
- 성격: 장비 속성 개선.
- 대표 효과: 검문/잠입/기업/보안 미션에서 발각 위험 감소 후보.
- 감소 원인 후보:
  - 위장 도색 적용.
  - 무광 코팅 적용.
  - 소음 억제 모듈 장착.
  - 열 차폐 패널 장착.
  - 신호 차폐 모듈 장착.
  - 장비 경량화.
  - 장비 소형화.
  - 외장 장식 제거.
  - 기업/군용 마킹 제거.
- 주요 변화 관계:
  - 기본 Visibility → Visibility 감소
  - Visibility 상승 → Visibility 감소
  - Visibility 감소 → 기본 Visibility
- 복구/정비 조건: 위장 개조 유지, 소음 억제 장치 정비, 차폐 모듈 점검.
- 상태 태그/임플란트 변화와 구분: 용병의 은밀함 상태가 아니라 장비 속성 변화다.
- 결과 보고서 표시 예시: `장비 노출도: 무광 코팅과 소음 억제로 낮게 유지.`
- 오용 금지: `#위장도색`, `#소음억제` 같은 장비 옵션을 용병 상태 태그로 올리지 않는다.

## 9. 정비/보험 계열

### 정비 필요

- 대상: 모든 외장 장비, 무기, 보호구, 도구.
- 성격: 비용 후보 변화.
- 대표 효과: 정비 전까지 기능 불안정 또는 비용 발생.
- 주요 변화 관계:
  - 오염/마모/퍼포먼스 저하/파손 → 정비 필요
  - 정비 필요 → 정상
- 복구/정비 조건: 크레딧 지불, 스테이션 정비, 부품 교체.
- 상태 태그/임플란트 변화와 구분: 정비 필요는 장비 상태다.
- 결과 보고서 표시 예시: `정비비: -180 크레딧.`
- 오용 금지: 치료비와 혼동하지 않는다.
- 플레이어 판독 단서: 이 항목은 장비 변화라기보다 정산/복구 안내에 가깝다.
- 다음 준비 판단: 실제 비용을 내고 복구할지, 위험을 감수하고 재출격할지 선택한다.
- 이동 후보: `결과 정산 항목 사전 1차` 작성 후 정산 항목으로 이동하거나 참조 처리한다.

### 보험 처리

- 대상: 보험 적용 장비, 외장형 무기, 드론, 보호구.
- 성격: 비용 완화 정산.
- 대표 효과: 수리비 감소, 영구 손실 복구 후보.
- 플레이어 판독 단서: 장비 손실이 사라지는 것이 아니라 비용 충격이 줄어든다.
- 다음 준비 판단: 고가 장비를 계속 굴릴지, 보험 없는 장비를 교체할지 판단한다.
- 주요 변화 관계:
  - 파손 → 보험 처리
  - 영구 손실 → 보험 보상 후보
- 복구/정비 조건: 사전 보험 가입, 계약 조건 충족.
- 상태 태그/임플란트 변화와 구분: 보험은 용병 상태가 아니라 정산 조건이다.
- 결과 보고서 표시 예시: `장비 보험 적용: 수리비 일부 감소.`
- 오용 금지: 보험 처리를 보상으로 과장하지 않는다. 손실 완화다.
- 이동 후보: `결과 정산 항목 사전 1차` 작성 후 정산 항목으로 이동하거나 참조 처리한다.

## 10. 1차 보류 변화

아래 후보는 장비 변화로 바로 채택하지 않는다.

```text
#호흡기_자극
#피로누적
#경미한_부상
#낮은_Frame
#높은_Visibility
임플란트 파츠 손상
임플란트 출력 저하
재시술 필요
```

보류 이유:

- 신체 증상 또는 상태 태그다.
- 스탯값을 태그화한다.
- `Visibility` 속성을 태그화한다.
- 임플란트 변화는 별도 사전에서 처리한다.
