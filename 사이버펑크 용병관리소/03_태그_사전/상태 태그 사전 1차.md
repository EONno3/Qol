# 상태 태그 사전 1차

이 문서는 실제 사용을 전제로 정리한 1차 상태 태그 사전이다.

중요:

- 이 문서는 최종 확정본이 아니다.
- 수치 효과는 아직 확정하지 않는다.
- 상태 태그는 용병 이력서의 `현재 상태`와 결과 보고서의 `상태 태그 변화`에 표시된다.
- 상태 태그는 스탯, 장비/임플란트 변화, 평판/노출 변화와 분리한다.

## 1. 작성 기준

상태 태그는 플레이어가 다음 판단을 하게 만드는 현재 조건이다.

기본 필드:

```text
태그명
주 분류
보조 분류
성격
대표 효과
주요 변화 관계
해소/완화/소모
결과 보고서 표기
오용 금지
```

상태 태그명은 UI에서 자연스럽게 읽히는 짧은 상태명으로 쓴다.

허용:

```text
#경미한_부상
#양호한_컨디션
#최상의_컨디션
#독성노출
```

금지:

```text
#피로도_상승
#Frame_저하
#침착유지
#의심증폭
#감각예민
```

## 2. 피로/회복 계열

### #피로누적

- 주 분류: 피로/회복.
- 보조 분류: 장시간 작업, 연속 출격.
- 성격: 부정 상태.
- 대표 효과: 다음 1회 장시간/정밀/환경 작업에서 집중도 저하 위험.
- 주요 변화 관계:
  - 없음 → `#피로누적`
  - `#피로누적` → `#심각한_피로누적`
  - `#피로누적` → 제거
- 해소/완화: 휴식, 식사, 스테이션 회복.
- 결과 보고서 표기: 원인과 다음 작업 영향 표시.
- 오용 금지: `피로 +1단계`, `#피로도_상승`처럼 게이지처럼 쓰지 않는다.

### #심각한_피로누적

- 주 분류: 피로/회복.
- 보조 분류: 장시간 작업, 회복 부족, 환경 부담.
- 성격: 강한 부정 상태.
- 대표 효과: 다음 1회 장시간/정밀/환경 작업에서 큰 불리 또는 돌발 노드 위험.
- 주요 변화 관계:
  - `#피로누적` → `#심각한_피로누적`
  - `#심각한_피로누적` → `#피로누적`
  - `#심각한_피로누적` → 제거
- 해소/완화: 긴 휴식, 치료, 스테이션 회복.
- 결과 보고서 표기: 기존 상태 악화/상위 전환으로 표시.
- 오용 금지: 단순 1회 피곤함에 바로 붙이지 않는다.

### #회복중

- 주 분류: 피로/회복.
- 보조 분류: 치료, 휴식.
- 성격: 조건부 상태.
- 대표 효과: 다음 출격 시 회복이 중단되거나 일부 미션에서 불리.
- 주요 변화 관계:
  - 부상/피로 상태 → `#회복중`
  - `#회복중` → 제거
  - `#회복중` → 기존 부정 상태 재발
- 해소/완화: 출격 없이 회복 시간 경과.
- 결과 보고서 표기: 치료 또는 휴식 반영 상태로 표시.
- 오용 금지: 완전히 회복된 상태를 뜻하지 않는다.

### #양호한_컨디션

- 주 분류: 피로/회복.
- 보조 분류: 긍정/조건부 컨디션.
- 성격: 긍정 상태.
- 대표 효과: 다음 1회 일반/정밀 작업에서 안정성 상승.
- 주요 변화 관계:
  - 없음 → `#양호한_컨디션`
  - `#양호한_컨디션` → `#최상의_컨디션`
  - `#양호한_컨디션` → 제거
- 해소/완화: 다음 출격 후 소모.
- 결과 보고서 표기: 긍정/조건부 상태 신규 부여로 표시.
- 오용 금지: 모든 미션 성공 보너스로 남발하지 않는다.

### #최상의_컨디션

- 주 분류: 피로/회복.
- 보조 분류: 긍정/조건부 컨디션.
- 성격: 강한 긍정 상태.
- 대표 효과: 다음 1회 일반/정밀/고위험 작업에서 안정성 상승.
- 주요 변화 관계:
  - `#양호한_컨디션` → `#최상의_컨디션`
  - `#최상의_컨디션` → `#양호한_컨디션`
  - `#최상의_컨디션` → 제거
- 해소/완화: 다음 출격 후 소모 또는 미션 부담으로 하위 전환.
- 결과 보고서 표기: 상위 전환 또는 강한 긍정 상태로 표시.
- 오용 금지: 장비/스테이션 준비를 대체하는 만능 버프로 쓰지 않는다.

## 3. 부상/신체 후유증 계열

### #경미한_부상

- 주 분류: 부상/신체 후유증.
- 보조 분류: 치료, 휴식.
- 성격: 약한 부정 상태.
- 대표 효과: 다음 1회 육체/추격/근접 대응 미션에서 불리.
- 주요 변화 관계:
  - 없음 → `#경미한_부상`
  - `#경미한_부상` → 제거
  - `#경미한_부상` → 더 강한 부상 상태 후보
- 해소/완화: 치료, 휴식.
- 결과 보고서 표기: 신규 부상으로 표시.
- 오용 금지: 임플란트 파손이나 장비 오염을 대신하지 않는다.

### #타박상

- 주 분류: 부상/신체 후유증.
- 보조 분류: 충돌, 추락, 근접전.
- 성격: 부정 상태.
- 대표 효과: 다음 1회 Frame 기반 육체 작업이나 근접 대응에서 불리.
- 주요 변화 관계:
  - 없음 → `#타박상`
  - `#타박상` → `#경미한_부상`
  - `#타박상` → 제거
- 해소/완화: 치료, 휴식.
- 결과 보고서 표기: 신체 충격 후유증으로 표시.
- 오용 금지: 단순 피로와 섞지 않는다.

### #출혈

- 주 분류: 부상/신체 후유증.
- 보조 분류: 의료, 생존 위험.
- 성격: 강한 부정 상태.
- 대표 효과: 다음 출격 제한 또는 치료 우선 필요.
- 주요 변화 관계:
  - 없음 → `#출혈`
  - `#출혈` → `#경미한_부상`
  - `#출혈` → 제거
- 해소/완화: 응급 처치, 치료, 의료 스테이션.
- 결과 보고서 표기: 즉시 치료 필요 상태로 표시.
- 오용 금지: 분위기 묘사용 피 묻음과 구분한다.

### #신경통

- 주 분류: 부상/신체 후유증.
- 보조 분류: 임플란트 부담, 신경계 압박.
- 성격: 부정 상태.
- 대표 효과: 다음 1회 정밀 조작, 반응, 집중 판정에서 불리.
- 주요 변화 관계:
  - 없음 → `#신경통`
  - `#신경통` → 제거
  - `#신경통` → 만성 문제 후보
- 해소/완화: 치료, 신경계 조정, 스테이션 안정화.
- 결과 보고서 표기: 신체 상태로 표시하되 임플란트 기능 저하와 분리.
- 오용 금지: 임플란트 파손 자체를 `#신경통`으로 대체하지 않는다.

### #시야차단

- 주 분류: 부상/신체 후유증.
- 보조 분류: 감각 상실, 임플란트 기능 상실 연계.
- 성격: 강한 부정 상태.
- 대표 효과: 시각 보조 기능 상실 상태로, 다음 1회 시각적 판단/조준/감시가 수반되는 미션에서 치명적인 불리.
- 주요 변화 관계:
  - 없음 (하드웨어 기능상실 시) → `#시야차단`
  - `#시야차단` → 제거 (하드웨어 수리/교체 시)
- 해소/완화: 연계된 시각 임플란트 파츠 수리 또는 교체 완료 시 제거 (휴식이나 자가 완화 없음).
- 결과 보고서 표기: 하드웨어 결함에 의한 시각 기능 상실 상태를 직접적으로 표시.
- 오용 금지: `#맹인`처럼 영구 장애를 뜻하지 않으며, 안구 가리개 등으로 인한 임시 장막과 분리한다.

### #청각차단

- 주 분류: 부상/신체 후유증.
- 보조 분류: 감각 상실, 임플란트 기능 상실 연계.
- 성격: 강한 부정 상태.
- 대표 효과: 청각 보조 기능 상실 상태로, 다음 1회 발소리 감지/상황 전파/무전 소통이 수반되는 미션에서 치명적인 불리.
- 주요 변화 관계:
  - 없음 (하드웨어 기능상실 시) → `#청각차단`
  - `#청각차단` → 제거 (하드웨어 수리/교체 시)
- 해소/완화: 연계된 청각 임플란트 파츠 수리 또는 교체 완료 시 제거 (휴식이나 자가 완화 없음).
- 결과 보고서 표기: 하드웨어 결함에 의한 청각 기능 상실 상태를 직접적으로 표시.
- 오용 금지: `#농인`처럼 영구 장애를 뜻하지 않으며, 소음 환경으로 인한 임시 청각 교란과 분리한다.

### #신경발열

- 주 분류: 부상/신체 후유증.
- 보조 분류: 바이오 거부 반응 연계, 면역학적 염증.
- 성격: 부정 상태.
- 대표 효과: 시냅스 및 접합부 면역 염증으로 인해 다음 1회 모든 스탯 기반 판정(특히 Cool, Pulse) 및 반응 지연 가능.
- 주요 변화 관계:
  - 없음 → `#신경발열`
  - `#신경발열` → 제거
- 해소/완화: 클리닉 치료, 면역 억제제 복용, 스테이션 집중 케어.
- 결과 보고서 표기: 바이오 임플란트 거부 반응으로 인한 전신 고열 및 신경계 스트레스를 표시.
- 오용 금지: 단순 고열 감기나 전투 중 입은 물리적 상처와 구분한다.

### #의체경련

- 주 분류: 부상/신체 후유증.
- 보조 분류: 바이오 거부 반응 연계, 근육 수축 오차.
- 성격: 부정 상태.
- 대표 효과: 바이오 근섬유 거부 반응으로 인해 다음 1회 Frame, Wire 기반 신체 기동 및 물리적 조작 판정에서 큰 불리.
- 주요 변화 관계:
  - 없음 → `#의체경련`
  - `#의체경련` → 제거
- 해소/완화: 클리닉 치료, 근이완 보조 조치, 휴식과 면역 조절.
- 결과 보고서 표기: 의체가 용병의 생체 신호를 거부하며 발생하는 불규칙한 신체 발작 및 경련 상태를 표시.
- 오용 금지: 신경 지연이나 단순 피로 누적과 구분한다.

## 4. 정신/압박 계열

### #긴장

- 주 분류: 정신/압박.
- 보조 분류: 협상, 검문, 제한 교전.
- 성격: 부정 또는 조건부 상태.
- 대표 효과: 다음 1회 협상/검문/제한 교전 미션에서 과잉 대응 위험.
- 주요 변화 관계:
  - 없음 → `#긴장`
  - `#긴장` → `#동요`
  - `#긴장` → 제거
- 해소/완화: 휴식, 상담, 유흥, 스테이션 안정화.
- 결과 보고서 표기: 압박 상황 후 신규 상태로 표시.
- 오용 금지: 성격 태그처럼 상시 성향으로 쓰지 않는다.

### #동요

- 주 분류: 정신/압박.
- 보조 분류: 실패 경험, 민간인 피해, 공포.
- 성격: 부정 상태.
- 대표 효과: 다음 1회 Cool 관련 안정 판정에서 불리.
- 주요 변화 관계:
  - `#긴장` → `#동요`
  - `#동요` → `#긴장`
  - `#동요` → 제거
- 해소/완화: 휴식, 상담, 성공 경험.
- 결과 보고서 표기: 압박 상태 악화로 표시.
- 오용 금지: 단순 놀람이나 짧은 반응에 붙이지 않는다.

### #각성

- 주 분류: 정신/압박.
- 보조 분류: 전투 직후, 경계, 집중.
- 성격: 조건부 상태.
- 대표 효과: 다음 1회 돌발 감지나 긴급 대응에 유리할 수 있으나, 협상/은밀 미션에서는 불리할 수 있음.
- 주요 변화 관계:
  - 없음 → `#각성`
  - `#각성` → `#흥분`
  - `#각성` → 제거
- 해소/완화: 휴식, 안정화, 시간 경과.
- 결과 보고서 표기: 조건부 상태로 표시.
- 오용 금지: 능력치 상승 버프처럼 고정 효과로 쓰지 않는다.

### #흥분

- 주 분류: 정신/압박.
- 보조 분류: 과잉 대응, 제한 교전 위험.
- 성격: 부정 또는 조건부 상태.
- 대표 효과: 다음 1회 제한 교전/검문/협상 미션에서 과잉 행동 위험.
- 주요 변화 관계:
  - `#각성` → `#흥분`
  - `#흥분` → `#각성`
  - `#흥분` → 제거

### #침착함

## 7. 1차 보류 태그

아래 후보는 상태 태그로 바로 채택하지 않는다.

```text
#호흡기_자극
#필터오염
#연결구불량
#소형도구소모
#근무리듬_안정
#낮은_Frame
#피로도_상승
#침착유지
#의심증폭
#감각예민
```

보류 이유:

- 장비/임플란트 변화와 섞인다.
- 스탯값을 태그화한다.
- 시스템 계산 과정처럼 보인다.
- UI 상태명으로 자연스럽지 않다.
