# MVP 결과 보고서 샘플

이 문서는 MVP 테스트 세트의 파견 후 결과 화면을 검증하기 위한 수동 샘플이다.

목적은 결과 문체를 최종 확정하는 것이 아니라, 아래 항목이 한 화면 안에서 플레이어에게 명확히 돌아오는지 확인하는 것이다.

```text
무슨 일이 일어났는가?
왜 성공/부분 성공/실패가 되었는가?
어떤 조건이 맞았고 무엇이 부족했는가?
보상, 손실, 상태 변화, 장비 변화, 평판 변화가 다음 선택으로 이어지는가?
```

---

## 1. 성공 샘플: 차단기 × 하층 변전소

### 요약 영역

- 미션명: 더 퓨즈 제3변전소 고압 캐패시터 절도
- 투입 용병: 백건우 / `차단기`
- 최종 결과: 성공
- 기본 보상: +16,500 크레딧
- 추가 보상: 없음
- 손실: 정비비 -700 크레딧
- 상태 태그 변화: 없음
- 장비 변화: 절연 안전화 외피 경미 마모
- 평판 변화: 더 퓨즈 세력 +5

### 사건 로그

차단기는 제3변전소 외곽 철망 앞에서 바로 진입하지 않았다.  
그는 바닥에 떨어진 피복 조각을 먼저 주워 냄새를 맡고, 과부하가 흐르는 선로를 우회했다.  
격실 내부에서 첫 번째 방전이 튀었지만, 고무 안전화 외피가 전류를 받아냈고 그는 작업을 멈추지 않았다.  
캐패시터 유닛 분리 과정에서 보조 관절이 한 번 삐걱였으나, 손목 토크를 낮춰 회로를 찢지 않고 빼냈다.  
철수 직전 낡은 차단기가 재가동됐지만, 그는 불꽃이 튄 방향과 반대로 몸을 빼며 손실 없이 빠져나왔다.

### 핵심 판정 요약

- 충족한 필수 조건: 절연 안전화 계열 기어 장착.
- 충족한 권장 조건: `Frame`, `Wire` 요구치 안정권.
- 성공을 만든 요소: 고압망 작업 경험, 현장 소리 판독, 절연 장비.
- 손실을 만든 요소: 구형 손목 토크 관절의 미세 부담.
- 조커 카드 발동: 없음.
- 돌발 노드 발생: 경미한 방전 노드 발생, 피해 없이 통과.

### 정산

- 보상: +16,500 크레딧
- 획득물: 마사다 고압 캐패시터 유닛 1개
- 추가 베네핏 미획득: 사이퍼 보너스 회로 다운 없음
- 손실: 절연 안전화 외피 정비비 -700 크레딧
- 상태 태그 변화: 없음
- 장비/임플란트 변화: 손목 토크 보조 관절 점검 필요
- 평판 변화: 더 퓨즈 세력 +5
- 후속 단서: 더 퓨즈 내부 전력 분배 장부의 절반이 캐패시터 포장지에 묻어 나왔다.

### 다음 반영

- 용병 이력서 현재 상태: 변화 없음.
- 장비 상태: 절연 안전화 `경미 마모` 기록.
- 팩션 평판: 더 퓨즈 세력 우호 소폭 상승.
- 후속 의뢰 후보: 전력 분배 장부 회수/해독 의뢰.

## 2. 부분 성공 샘플: 몰트 × 중층 클럽

### 요약 영역

- 미션명: 클럽 엘 네온 백도어 강탈
- 투입 용병: 한서윤 / `몰트`
- 최종 결과: 부분 성공
- 기본 보상: +24,000 크레딧
- 추가 보상: 없음
- 손실: 채무 중재금 -3,000 크레딧
- 상태 태그 변화: `#수면부족` 심화
- 장비 변화: 손목형 인증 우회기 과열
- 평판 변화: 픽서 네트워크 +3, 라 나다 -3

### 사건 로그

몰트는 클럽 입구에서 취객처럼 웃으며 줄을 잘못 선 척했다.  
키쿠모토식 안면 식별 스캐너는 그녀의 얼굴을 한 번 더 훑었지만, 손목 팔찌형 우회기가 지나간 결제 기록을 덮어씌웠다.  
VIP 룸 앞에서는 가드가 그녀를 알아본 듯 멈칫했다.  
몰트는 대답 대신 오래된 술집 이름을 낮게 말했고, 가드는 총 대신 손을 내밀었다.  
암호 칩은 회수했지만, 그 자리에서 채무 중재금 일부를 떼어 줘야 했다.  
철수 후 우회기는 손목 안쪽에 열 자국을 남겼고, 그녀는 보고서 작성 중에도 두 번이나 잠깐 의식을 놓쳤다.

### 핵심 판정 요약

- 충족한 필수 조건: 장비 `Visibility` 기준 이하.
- 충족한 권장 조건: `Cypher`, `Cool` 요구치 충족.
- 성공을 만든 요소: 시민 위장, 스캐너 기만, 은닉형 슬롯.
- 손실을 만든 요소: 채무 관계, 우회기 과열, 누적 수면 부족.
- 조커 카드 발동: 없음.
- 돌발 노드 발생: 갱단 인맥 확인 노드 발생, 크레딧 지불로 통과.

### 정산

- 보상: +24,000 크레딧
- 획득물: 암호화 백도어 코어 세트
- 추가 베네핏 미획득: VIP 홀 잔여 거래 데이터 회수 실패
- 손실: 채무 중재금 -3,000 크레딧, 우회기 냉각 정비비 -1,200 크레딧
- 상태 태그 변화: `#수면부족` → `#수면부족_심화`
- 상태 태그 효과: 다음 1회 미션에서 장기 대기/잠복 판정 시 집중력 저하 위험.
- 장비/임플란트 변화: 손목형 근거리 인증 우회기 `과열 흔적`
- 평판 변화: 픽서 네트워크 +3, 라 나다 -3
- 후속 단서: 라 나다 장부에 몰트의 예전 술집 빚이 아직 살아 있다.

### 다음 반영

- 용병 이력서 현재 상태: `#수면부족_심화` 추가.
- 장비 상태: 손목형 인증 우회기 냉각 정비 필요.
- 팩션 평판: 라 나다와의 관계 악화.
- 후속 의뢰 후보: 몰트의 채무 장부 삭제 의뢰.

## 3. 실패 샘플: 크롬쇼 × 중층 클럽

### 요약 영역

- 미션명: 클럽 엘 네온 백도어 강탈
- 투입 용병: 도마 준 / `크롬쇼`
- 최종 결과: 실패
- 기본 보상: 0 크레딧
- 추가 보상: 없음
- 손실: 장비 수리비 -6,500 크레딧
- 상태 태그 변화: `#어깨장갑판_균열` 악화, `#탄환_관통흔` 추가
- 장비 변화: 고출력 산탄 암 과열, 노출형 장갑판 파손
- 평판 변화: 라 나다 -6, 픽서 네트워크 -2

### 사건 로그

크롬쇼는 클럽 정문 앞에서 이미 군중의 시선을 끌고 있었다.  
도금된 어깨 장갑판이 네온 조명을 받아 번쩍였고, 키쿠모토 스캐너는 그의 팔 안쪽 산탄 암을 민간 장비로 처리하지 않았다.  
첫 번째 경고음이 울리자 그는 대기 명령을 무시하고 가드의 손목을 잡아 꺾었다.  
셔터가 내려가고, VIP 룸까지 이어지는 통로가 교전 구역으로 바뀌었다.  
그는 살아서 빠져나왔지만, 암호 칩은 회수하지 못했고 장갑판에는 총알과 스파크 자국이 함께 남았다.  
철수 차량에 오른 뒤에도 산탄 암은 식지 않았고, 정비공은 팔을 분리하지 않으면 화재가 난다고 말했다.

### 핵심 판정 요약

- 충족한 필수 조건: 없음.
- 미충족한 필수 조건: 장비 `Visibility` 기준 초과.
- 충족한 권장 조건: 정면 교전 생존력.
- 미충족한 권장 조건: `Cypher`, `Cool`, 은닉형 세팅.
- 실패를 만든 요소: 과시형 크롬, 높은 가시성, 대기명령 불복.
- 조커 카드 발동: 생존성은 작동했으나 미션 목표를 보존하지 못함.
- 돌발 노드 발생: 진입 즉시 검문 적발, 갱단 교전 노드 강제 발생.

### 정산

- 보상: 0 크레딧
- 획득물: 없음
- 추가 베네핏 미획득: 암호 칩 회수 실패, 거래 데이터 회수 실패
- 손실: 장비 수리비 -6,500 크레딧
- 상태 태그 변화: `#어깨장갑판_균열` 악화, `#탄환_관통흔` 추가
- 상태 태그 효과: 다음 미션에서 가시성 상승, 장시간 전투 시 통증 반응 증가.
- 장비/임플란트 변화: 고출력 산탄 암 과열, 노출형 장갑판 파손
- 평판 변화: 라 나다 -6, 픽서 네트워크 -2
- 후속 단서: 라 나다 쪽에서 크롬쇼의 얼굴과 팔 형상을 현상금 보드에 올렸다.

### 다음 반영

- 용병 이력서 현재 상태: `#탄환_관통흔`, 장갑판 파손 반영.
- 장비 상태: 고출력 산탄 암 정비 전 사용 제한.
- 팩션 평판: 라 나다 적대 상승.
- 후속 의뢰 후보: 라 나다 현상금 삭제 또는 보복 미션.

## 4. MVP 결과 검증 포인트

- 성공 샘플은 매칭의 유리 신호가 실제 사건 로그와 정산으로 이어지는지 확인한다.
- 부분 성공 샘플은 미션 목표는 달성했지만 경제/상태/관계 손실이 남는지 확인한다.
- 실패 샘플은 출격 전 경고 또는 불가 판정을 무시했을 때 결과 보고서가 같은 원인을 되돌려 주는지 확인한다.
- 모든 샘플은 결과가 용병 이력서, 장비 상태, 평판, 후속 의뢰 후보 중 최소 하나에 반영되어야 한다.
