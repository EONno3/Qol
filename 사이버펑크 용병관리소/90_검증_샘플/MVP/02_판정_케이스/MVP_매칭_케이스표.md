# MVP 매칭 케이스표

이 문서는 `90_검증_샘플/MVP/00_개요/MVP_테스트_세트.md`의 미션 3개와 `90_검증_샘플/MVP/01_입력_데이터/MVP_테스트_용병_이력서.md`의 용병 4명을 서로 대입해, 미션 슬롯 매칭 규칙이 실제로 작동하는지 검증하기 위한 케이스표다.

목적은 최종 밸런스 확정이 아니라 아래 질문에 답하는 것이다.

```text
미션 카드와 용병 이력서를 읽고 판단할 수 있는가?
분석 레벨이 낮을 때 정보가 제대로 숨겨지는가?
분석 레벨이 올라가면 유리/불리/조건부 신호가 자연스럽게 드러나는가?
출격 판정과 결과 보고서가 같은 이유를 공유하는가?
```

---

## 1. 공통 매칭 레벨 규칙

모든 케이스는 아래 규칙을 따른다.

```text
매칭 표시 레벨 = min(용병 분석 레벨, 미션 분석 레벨)
```

- 매칭 Lv.0: 매칭 신호 없음. 용병 이력서와 미션 카드의 독립 정보만 확인한다.
- 매칭 Lv.1: 유리 / 불리 / 조건부 / 미확인 신호와 거친 성공 전망을 표시한다.
- 매칭 Lv.2: 충돌/시너지 이유, 예상 성공률 방향, 예상 정산 손익, 결과 보고서 확인 포인트를 표시한다.

## 2. 하층 미션: 더 퓨즈 제3변전소 고압 캐패시터 절도

- 기준 문서: `04_미션/04_하층_변전소_캐패시터_탈취.md`
- 핵심 축: `Frame`, `Wire`
- 필수 기어: 절연 안전화 계열 기어
- 주요 위험: 감전, 전뇌 단락, 의체 마손, 정산금 감산

### 2.1 차단기 대입

- 출격 판정: 가능
- 매칭 Lv.1 신호:
  - 유리: 산업 설비 경험, 고압망 작업 습관.
  - 불리: 보안 절차 무시 경향.
  - 조건부: 절연 장비 상태가 온전해야 안정적이다.
  - 미확인: 과거 감전 사고 기록 일부.
- 매칭 Lv.2 해석:
  - `#고압망_기술`과 `Frame/Wire` 수치가 미션 요구와 맞물린다.
  - `#감전_잔향`은 실패 시 상태 악화 후보로 남는다.
  - 예상 방향: 성공 우세. 장비 파손 비용만 결과 보고서에서 확인한다.
- 결과 보고서 확인 포인트: 절연 장비 내구도, 감전 상태 전이 여부, 캐패시터 손상 여부.

### 2.2 몰트 대입

- 출격 판정: 경고
- 매칭 Lv.1 신호:
  - 유리: 조용한 동선, 낮은 가시성.
  - 불리: 산업 설비 직접 처리 능력 부족.
  - 조건부: 별도 절연 기어를 지급하면 출격 자체는 가능하다.
  - 미확인: 채무 관계가 현장 변수로 개입할 가능성.
- 매칭 Lv.2 해석:
  - `Cypher` 강점은 현장 은닉에는 도움되지만 변압기 해체의 핵심 축은 아니다.
  - `Frame/Wire` 요구를 안정적으로 넘기지 못해 작업 지체 가능성이 높다.
  - 예상 방향: 부분 성공 또는 정산금 감산 위험.
- 결과 보고서 확인 포인트: 감전 회피 여부, 작업 지체로 인한 보상 감산, 외부 추적 흔적.

### 2.3 벨벳 나이프 대입

- 출격 판정: 경고
- 매칭 Lv.1 신호:
  - 유리: 낮은 가시성, 침착한 행동.
  - 불리: 하층 산업 현장과 물리 작업 경험 부족.
  - 조건부: 보호 장비를 맞추면 현장 진입은 가능하다.
  - 미확인: 반사면 지연 반응이 설비 경고등 앞에서 문제될 수 있다.
- 매칭 Lv.2 해석:
  - `Cool/Cypher`는 경보 대응에는 도움되지만 핵심 해체 판정과 어긋난다.
  - 고급 위장 소모품은 하층 전력망 환경에서 비용 대비 효율이 낮다.
  - 예상 방향: 성공 가능은 있으나 손익 비효율.
- 결과 보고서 확인 포인트: 소모품 낭비, 장비 오염, 고급 위장 장비 손상.

### 2.4 크롬쇼 대입

- 출격 판정: 경고
- 매칭 Lv.1 신호:
  - 유리: 높은 프레임, 물리 돌파력.
  - 불리: 전도성 크롬, 누전 과민, 낮은 와이어.
  - 조건부: 절연 장비 없이는 치명적 손상 위험.
  - 미확인: 신경계 과민 반응.
- 매칭 Lv.2 해석:
  - `Frame`은 충분하지만 `#누전_과민`과 전도성 장갑판이 미션 위협과 정면 충돌한다.
  - 절연 장비를 착용해도 장비 마손과 상태 손상 위험이 높다.
  - 예상 방향: 출격은 가능하나 실패/부상 유도 케이스로 적합.
- 결과 보고서 확인 포인트: `#전뇌_단락_충격`, 의체 내구도 마손, 정산금 -4,000 감산 여부.

## 3. 중층 미션: 클럽 엘 네온 백도어 강탈

- 기준 문서: `04_미션/02_중층_클럽_엘네온_강탈.md`
- 핵심 축: `Cypher`, `Cool`
- 필수 조건: 장비 `Visibility` 총합 40 이하
- 주요 위험: 검문 적발, 갱단 교전, 장비 파손

### 3.1 차단기 대입

- 출격 판정: 경고
- 매칭 Lv.1 신호:
  - 유리: 낮은 가시성, 침착한 손작업.
  - 불리: 클럽 보안망과 사회적 위장 경험 부족.
  - 조건부: 저가시성 장비만 유지하면 진입 가능.
  - 미확인: 보안 절차 무시 성향.
- 매칭 Lv.2 해석:
  - `Wire`는 일부 장치 처리에 도움되지만 핵심인 `Cypher/Cool`이 부족하다.
  - 작업자 같은 태도가 클럽 내부에서는 위장 노이즈가 될 수 있다.
  - 예상 방향: 진입 후 후반 리스크 증가.
- 결과 보고서 확인 포인트: 검색 게이트 통과, 경보 상승, 암호 칩 회수 실패 여부.

### 3.2 몰트 대입

- 출격 판정: 가능
- 매칭 Lv.1 신호:
  - 유리: 시민 위장, 스캐너 기만, 낮은 가시성.
  - 불리: 채무 관계로 인한 갱단 변수.
  - 조건부: 은닉형 슬롯과 저가시성 무장 유지 필요.
  - 미확인: 특정 술집 장부와의 연결.
- 매칭 Lv.2 해석:
  - `#시민_위장`, `#스캐너_기만`, `#은닉형_슬롯`이 미션 요구와 직접 맞물린다.
  - `Cool 49`, `Cypher 56`으로 핵심 요구를 안정적으로 넘긴다.
  - 예상 방향: 성공 우세. 단, 채무 이벤트가 결과 보고서 변수로 남는다.
- 결과 보고서 확인 포인트: 검문 통과, 갱단 경보 미발생, 채무 단서 발생 여부.

### 3.3 벨벳 나이프 대입

- 출격 판정: 가능
- 매칭 Lv.1 신호:
  - 유리: 매우 낮은 가시성, 침착한 위장.
  - 불리: 상층식 행동 양식이 중층 클럽에서 이질적으로 보일 수 있다.
  - 조건부: 과한 고급 위장 장비를 줄이면 안정적이다.
  - 미확인: 봉인 계약 관련 추적 흔적.
- 매칭 Lv.2 해석:
  - `Cool/Cypher`는 충분하지만, 이 미션에서는 과잉 투입에 가깝다.
  - 보상 대비 신원 삭제 비용과 소모품 비용이 커질 수 있다.
  - 예상 방향: 성공 가능 높음, 순이익 효율 낮음.
- 결과 보고서 확인 포인트: 성공 보상 대비 추가 비용, 상층 신원 흔적 노출 여부.

### 3.4 크롬쇼 대입

- 출격 판정: 불가
- 매칭 Lv.1 신호:
  - 유리: 교전 발생 시 생존력.
  - 불리: 매우 높은 가시성, 노출형 크롬, 낮은 `Cypher/Cool`.
  - 조건부: 장비를 대폭 해체하지 않으면 진입 불가.
  - 미확인: 흥분도와 대기명령 불복.
- 매칭 Lv.2 해석:
  - `Visibility 매우 높음`이 필수 조건과 정면 충돌한다.
  - 출격시키면 진입 단계에서 즉시 발각 분기로 전이된다.
  - 예상 방향: MVP 실패 유도 케이스.
- 결과 보고서 확인 포인트: 진입 차단, 경보 폭증, 장비 파손, 라 나다 평판 손실.

## 4. 상층 미션: 알-마디낫 후계자 전뇌 말소

- 기준 문서: `04_미션/03_상층_펜트하우스_스파이.md`
- 핵심 축: `Cool`, `Cypher`
- 필수 기어: 상층 고고도 기압 보정기, 인공 DNA 유전자 스킨 패치
- 주요 위험: 바이오메트릭 적발, 드론 요격, 상층 평판 붕괴, 사망 확률 연산

### 4.1 차단기 대입

- 출격 판정: 불가
- 매칭 Lv.1 신호:
  - 유리: 침착한 손작업 일부.
  - 불리: 상층 예법/바이오 위장/사교 대응 부족.
  - 조건부: 필수 기어 2종이 없으면 출격 불가.
  - 미확인: 하층 작업자 이력이 신원 대조에서 잡힐 가능성.
- 매칭 Lv.2 해석:
  - `Cool/Cypher` 요구를 만족하지 못하고 상층 필수 기어도 기본 보유가 아니다.
  - 산업 현장 강점은 펜트하우스 위장 침투와 거의 맞물리지 않는다.
  - 예상 방향: 출격 차단이 정상.
- 결과 보고서 확인 포인트: 실제 출격시키지 않는 것이 테스트 통과 조건.

### 4.2 몰트 대입

- 출격 판정: 경고
- 매칭 Lv.1 신호:
  - 유리: 낮은 가시성, 출입 관리 경험.
  - 불리: 상층 예법과 DNA 위장 장비 부족.
  - 조건부: 필수 기어 2종을 임시 지급하면 진입 가능성이 생긴다.
  - 미확인: 중층 채무 흔적이 상층 신원 세탁에 방해될 수 있다.
- 매칭 Lv.2 해석:
  - `Cypher`는 충분하지만 `Cool`과 상층식 행동 노이즈가 불안정하다.
  - 장비를 맞춰도 사교 연회장 판정에서 위험이 남는다.
  - 예상 방향: 고위험 조건부 출격.
- 결과 보고서 확인 포인트: 바이오게이트 통과, 사교 예법 실패, 추적 비용.

### 4.3 벨벳 나이프 대입

- 출격 판정: 가능
- 매칭 Lv.1 신호:
  - 유리: 유전자 기만, 단정한 인상, 상층 예법.
  - 불리: 반사면 지연 반응.
  - 조건부: 상층 기압 보정기와 DNA 패치가 완비되어야 한다.
  - 미확인: 봉인된 상층 계약 기록.
- 매칭 Lv.2 해석:
  - `Cool 64`, `Cypher 58`, `#유전자_기만`, `#단정한_인상`이 핵심 요구와 맞물린다.
  - `#반사면_지연반응`은 바이오 감시망 앞의 잔여 리스크로 남는다.
  - 예상 방향: 성공 우세. 비용은 높지만 상층 미션 기준으로 정당화된다.
- 결과 보고서 확인 포인트: 바이오게이트 통과, 반사면 리스크 발현 여부, 신원 삭제 비용.

### 4.4 크롬쇼 대입

- 출격 판정: 불가
- 매칭 Lv.1 신호:
  - 유리: 직접 교전 생존력.
  - 불리: 높은 가시성, 낮은 `Cool/Cypher`, 하층 슬랭 고착.
  - 조건부: 장비와 외형을 거의 전부 교체해야 한다.
  - 미확인: 흥분도와 상층 경비 반응.
- 매칭 Lv.2 해석:
  - 필수 위장 조건, 예법, 바이오메트릭 대응을 모두 만족하지 못한다.
  - 출격시키면 미션 테스트가 아니라 즉시 요격/사망 판정 테스트가 된다.
  - 예상 방향: 출격 차단이 정상. 강제 출격 시 최악 실패 샘플.
- 결과 보고서 확인 포인트: 출격 차단 UI, 강제 출격 예외 처리, 사망 확률 연산.

## 5. MVP 검증 결론

이 케이스표에서 반드시 확인해야 하는 결과는 아래와 같다.

- 적합 용병 3명은 각 대표 미션에서 `가능` 또는 안정적 `경고` 판정을 받아야 한다.
- 크롬쇼는 중층/상층 침투 미션에서 `불가` 또는 치명적 `경고`를 받아야 한다.
- 차단기는 하층에서는 강하지만 중층/상층에서는 장점이 제한적으로만 작동해야 한다.
- 몰트는 중층에서는 강하지만 상층에서는 장비와 예법 조건 때문에 `경고`가 떠야 한다.
- 벨벳 나이프는 상층에서는 강하지만 하층/중층에서는 비용 대비 과잉 투입 문제가 드러나야 한다.
- 매칭 Lv.0에서는 위 해석이 직접 표시되지 않고, Lv.1/2에서만 단계적으로 드러나야 한다.
