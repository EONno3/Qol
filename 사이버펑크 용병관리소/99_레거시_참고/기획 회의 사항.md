# 기획 회의 사항

> 문서 역할: 이 문서는 태그 시스템의 초기 브레인스토밍/회의 메모다.
>
> 최신 기준: 실제 태그 작성 기준은 `03_태그_사전/태그 시스템 기본 구조.md`, `03_태그_사전/규칙/시스템 태그 규칙.md`, `03_태그_사전/규칙/이야기 태그 규칙.md`, `03_태그_사전/규칙/미션 태그 해석 규칙.md`, `03_태그_사전/규칙/태그 분류 규칙.md`를 우선한다.
>
> 주의: 아래의 `태그명/발동 트리거/시스템적 결과` 방식은 초기 아이디어 정리용이다. 최신 기준에서는 태그 자체를 단순하게 두고, 복잡한 효과와 수치 해석은 미션/이야기/장비 규칙에서 처리한다.

**1. 태그(Tag) 시스템의 도입 목적 (Why we need this)** 우리 게임은 픽서(플레이어)가 텍스트와 수치를 보고 용병을 파견하는 매니지먼트 게임입니다. 스탯(프레임, 쿨 등)과 장비(임플란트, 무기)가 용병의 **'예측 가능한 성능'**을 보여준다면, 특이사항 태그는 예측을 비틀어버리는 **'돌발 변수'**를 담당합니다.

- **목표 1:** A급 스탯을 가진 용병이라도 치명적인 단점 태그 때문에 파견을 망설이게 만들기.
    
- **목표 2:** C급 스탯의 폐급 용병이라도 특정 태그가 미션과 맞아떨어지면 조커 카드로 쓰일 수 있게 만들기.
    
- **결론:** 픽서의 뇌를 괴롭히고(경영의 재미), 텍스트 결과 보고서에 드라마(서사)를 만들어내기 위함.
    

**2. 태그가 개입하는 4가지 핵심 시스템 노드 (Where it hooks)** 태그 아이디어를 낼 때, 단순히 "얘는 성격이 포악함"으로 끝내면 안 됩니다. 우리가 만든 기존 시스템(장비, 스탯, 미션, 경영) 중 최소 하나를 건드려야 합니다. 회의하실 때 아래 4가지 분야 중 어디에 속하는 변수일지 고민해 보시면 좋습니다.

- **A. 하드웨어 세팅 변수 (임플란트 & 무기)**
    
    - _개념:_ 장비를 낄 때 비용이나 효율을 건드리는 태그.
        
    - _브레인스토밍 방향:_ "크롬은 기가 막히게 잘 받는데, 내장 무기만 달면 고장 내는 놈은 어떨까?", "구형(싸구려) 임플란트만 껴야 오히려 전투력이 오르는 변태 같은 특성은?"
        
- **B. 스탯 및 주사위 굴림 변수 (전투 & 잠입)**
    
    - _개념:_ 특정 미션 상황에서 5대 스탯(프레임, 쿨, 와이어, 사이퍼, 펄스)의 수치를 뻥튀기하거나 깎아버리는 태그.
        
    - _브레인스토밍 방향:_ "평소엔 쿨(Cool)이 바닥인데, 자기 피통이 10% 미만이 되면 쿨과 펄스가 폭주하는 각성 태그?", "비밀 미션에서만 와이어 스탯이 0으로 고정되는 기계치 특성?"
        
- **C. 세계관 및 상성 변수 (미션 진영)**
    
    - _개념:_ 메가코프, 뒷골목, 특정 의뢰인 등 진영 평판이나 특정 미션(지원/비밀/기업 등)에 진입할 때 작용하는 태그.
        
    - _브레인스토밍 방향:_ "특정 메가코프 미션에 보내면 보상을 2배로 물어오지만, 실패하면 픽서 사무소로 사병들이 쳐들어올 확률이 생기는 내부고발자 태그?"
        
- **D. 경영 및 유지비 변수 (바/식당 & 크레딧)**
    
    - _개념:_ 전투가 끝난 후, 픽서의 지갑(크레딧/플럭스)과 용병의 멘탈(계약 이행률)을 건드리는 태그.
        
    - _브레인스토밍 방향:_ "미션 보수는 안 받는데 바(Bar)에서 술을 안 사주면 무단이탈하는 놈?", "임무 완료 시 파손된 적의 무기를 주워와서 크롬(재화)으로 환전해 주는 알뜰한 태그?"
        

**3. 팀원들과 태그를 만들 때 사용할 [아이디어 템플릿]** 회의에서 아이디어가 중구난방으로 퍼지지 않도록, 태그를 제안할 때는 다음 3가지 요소를 세트로 묶어서 제시하자는 규칙을 세우면 좋습니다.

1. **태그명:** (예: `#사이버싸이코_경계선`)
    
2. **발동 트리거 (조건):** (예: 임플란트 코스트를 80% 이상 채웠을 때)
    
3. **시스템적 결과 (Effect):** (예: 전투 미션 성공률 +30%, 단 임무 완료 후 아군 NPC를 폭행하여 위약금 발생 확률 20%)

무기 스탯
	화력 / 명중률 / 내구도 / 비용(구매/보험) 

임플란트 태그 종류
태그 종류
	기능(매칭) / 미션 특화 (대상, 목적등) / 스탯 특화 / 이벤트 특화 / 임플란트 고유 특화 / 세트 효과(임플란트의 시리즈별 세트효과 등)
스탯
	화력(무기형의 경우) / 내구도 / 코스트 / 비용(구매/보험)

    부정
        기본 태그 / 비용 및 코스트 / 범죄 / 상태(가변) / 특징
    긍정
        기본 태그 / 비용 및 코스트 / 명성 / 상태(가변) / 특징
