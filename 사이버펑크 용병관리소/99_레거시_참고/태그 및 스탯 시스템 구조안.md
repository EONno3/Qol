# 태그 및 스탯 시스템 구조안

> 문서 역할: 이 문서는 무기/임플란트/용병 태그와 스탯의 초기 구조안이다.
>
> 최신 기준: 실제 태그 생성과 분류는 `03_태그_사전/태그 시스템 기본 구조.md`, `03_태그_사전/태그 작성 양식.md`, `03_태그_사전/규칙/시스템 태그 규칙.md`, `03_태그_사전/규칙/이야기 태그 규칙.md`, `03_태그_사전/규칙/미션 태그 해석 규칙.md`, `03_태그_사전/규칙/태그 분류 규칙.md`를 우선한다.
>
> 주의: 아래의 `미션 특화`, `스탯 특화`, `이벤트 특화`, `긍정/부정` 분류는 레거시 참고 자료다. 최신 기준에서는 태그를 시스템/이야기, 공용/전용, 주 분류 10개 축으로 다시 판정한다.

무기
	태그 종류
		기능 : 해당 무기가 수행하는 가장 기본적인 형태와 역할 (예: 실탄 소총, 스마트 런처).
		매칭 : 무기의 기본 기능 외에, 특정 임플란트(예: 시각 보조 장치)와 결합했을 때 추가적인 시너지를 활성화하는 조건부 태그.
		미션 특화 : 특정 타겟(예: 중장갑, 드론)이나 목적(예: 암살, 전면전)에 투입되었을 때 명중률이나 가시성 패널티를 보정해 주는 지표.
		스탯 특화 : 장비를 장착하기 위해 특정 스탯 수치를 강제하거나, 장착 시 용병의 특정 스탯에 보너스/패널티를 부여하는 기믹.
		이벤트 특화 : 미션 진행 중 '적의 기습', '검문소 통과' 등 특정 돌발 텍스트 상황이 발생했을 때 이를 회피하거나 특수 로그를 띄우는 변수.
		무기 고유 특화 : '도탄', '벽 투시 타격', 'EMP 발생' 등 다른 무기 그룹에는 없는 해당 장비만의 고유한 작동 방식과 특수 스킬.
	무기 스탯
		화력: 미션 성공률 및 교전 소요 시간(시간 단축)에 관여하는 기본 데미지 수치.
		명중률: 화력을 온전히 투사하기 위한 확률값. 미달 시 미션 실패 굴림에 영향을 줌.
		내구도: 미션 중 마모되는 무기의 체력. 0이 될 경우 파손되어 페널티가 발생함. (수리비 산정의 기준점)
		비용 (구매/보험): 상점 구매가 및 파견 시 픽서가 선택할 수 있는 리스크 관리 비용. 
			- 보험 가입 시: 파손되어도 수리/복구 가능. (유지비 증가, 리스크 감소)
			- 보험 미가입 시: 유지비는 아끼지만 파손 시 영구 삭제(복구 불가) 처리.

임플란트
	태그 종류
		기능 : 신체를 대체하거나 보조하는 본연의 역할 (예: 호흡 필터, 안구).
		매칭 : 무기나 타 임플란트와 짝을 이루어 시너지를 해금하는 조건부 태그.
		미션 특화 : 수중, 독가스, 전뇌 공간 등 특수 환경 미션에서 강제되거나 생존율을 대폭 보정하는 지표.
		스탯 특화 : 용병의 5대 코어 스탯(프레임, 쿨, 와이어, 사이퍼, 펄스)을 증감시키는 지표.
		이벤트 특화 : 사망 직전 출혈 발생 시 '자극제 투여' 등 위기 상황을 시스템적으로 모면하게 해주는 히든 변수.
		임플란트 고유 특화 : 동일한 '기능'을 공유하는 제품군 안에서의 개별 모델 특징. (예: 똑같은 '호흡 필터' 기능이더라도 A모델은 [독가스 재생산], B모델은 [잉여 산소 저장]으로 바리에이션이 나뉨).
		세트 효과 : 동일한 제조사(예: 밀리테크 시리즈)나 라인업의 파츠를 다수 장착했을 때 추가로 활성화되는 컬렉션 보너스.
	임플란트 스탯
		화력 (무기형 한정): 투사체 런처, 암 커터 등 공격 기능이 내장된 파츠일 경우 적용되는 데미지 수치.
		내구도: 임플란트의 자체 체력. 손상 시 용병의 신체 결함 및 페널티 발생.
		코스트: 용병의 기본 프레임/신경계가 감당할 수 있는 한도 내에서 해당 파츠가 차지하는 장착 용량.
		비용 (구매/보험): 무기와 동일하게 구매가 및 파손 영구 삭제를 막기 위한 선택적 보험료.

용병
가질수 있는 태그 칸의 제한은 딱히 없으나 태생 3 4 5 개 이런느낌으로 뽑힐때 부터 가지고 있는 태그 수의 차이는 있음 - 물론 여기에 부정 긍정 태그가 섞여 있음
기본 태그 : 출신지, 외형 등등 -5개정도
