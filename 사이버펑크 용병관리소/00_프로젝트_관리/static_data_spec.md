## 1. 출신지(Origin) 7종 효과 테이블

게임 시작 시 선택. 이후 변경 불가.

### 설계 원칙
- 출신지는 **초기 기울기**만 결정한다. 플레이 중 어느 방향으로든 바꿀 수 있다.
- 수치는 방향성 기준값이며, 밸런스 튜닝 시 조정 가능.
- 모든 출신지는 **장점과 단점을 동시에** 가진다.

---

### 1.1 하층 언존 태생 (Lower Unzone Native)

> 돔 최하층 슬럼(UNzone) 출신. 태어날 때부터 법도 보호도 없는 곳에서 자랐다.

| 항목 | 값 |
|---|---|
| 초기 크레딧 | 1,500 (낮음) |
| 배경 태그 | `#슬럼_출신` |
| 스테이션 초기 위치 | 하층만 선택 가능 |
| 하층 팩션 전체 호감도 | **+20** |
| 중층 팩션 전체 호감도 | -5 |
| 상층/메가코프 호감도 | -10 |

**`#슬럼_출신` 태그 효과:**
- 하층 미션: 로컬 네트워크 접근 용이 (특정 진입 노드 우회 가능)
- 중층 미션: 신분 검문 시 위화감 발생 확률 ↑ (복장/언어)
- 상층 미션: 바이오스캔 시 즉각 이질 판정 위험

---

### 1.2 살아남은 폐기체 (Surviving Discarded Specimen)

> 기업 실험을 위해 양산된 복제인간. 폐기처리됐으나 생존해 언존 쓰레기 더미 속에서 눈을 떴다. 신원이 없다. 기록이 없다.

| 항목 | 값 |
|---|---|
| 초기 크레딧 | 0 (없음, 신원 자체가 없음) |
| 배경 태그 | `#신원_말소`, `#실험체_이력` |
| 스테이션 초기 위치 | 하층만 선택 가능 |
| 바이루이 호감도 | **-40** (실험한 기업) |
| 더 널스(UNzone) 호감도 | **+15** (같은 쓰레기 더미 출신) |
| 나머지 팩션 호감도 | ±0 |

**`#신원_말소` 태그 효과:**
- 신분 검문 시 데이터베이스에 기록 없음 → 즉각 요주의 대상 분류 위험
- 위장 신분 조달 비용 증가 (기록 자체가 없으니 위조가 더 어려움)
- 특정 기업 미션에서 역으로 유리: 추적 불가한 존재이므로 익명 작전에 선호됨

**`#실험체_이력` 태그 효과:**
- 바이루이 관련 미션에서 특수 이야기 이벤트 트리거 가능
- 임플란트 코스트 일부 감소 (이미 실험적 개조가 된 몸)
- 고도화: 기억 단편 회복 이벤트 트리거 조건

---

### 1.3 버려진 사생아 (Abandoned Bastard)

> 상층 기업 혈통이지만 공식 인정받지 못한 존재. 어느 계층에도 완전히 속하지 않는다.

| 항목 | 값 |
|---|---|
| 초기 크레딧 | 3,500 (보통) |
| 배경 태그 | `#혈연_연줄` |
| 스테이션 초기 위치 | 하층 / 중층 선택 가능 |
| 선택한 메가코프 1개 호감도 | **+15** (혈연 기업) |
| 나머지 메가코프 호감도 | ±0 |
| 하층 팩션 호감도 | ±0 |

> 게임 시작 시 어느 메가코프 혈통인지 1개 선택 (바이루이 / 마사다 / 키쿠모토 / 알-마디낫).

**`#혈연_연줄` 태그 효과:**
- 선택 기업 관련 미션에서 특수 선택지 발생 가능 (내부 정보 접근, 관계자 접선)
- 해당 기업 적대 미션 수행 시 추가 평판 패널티 위험

---

### 1.4 추방자 (Exile)

> 특정 팩션이나 구역에서 쫓겨난 자. 연줄은 있지만 이미 불태웠다.

| 항목 | 값 |
|---|---|
| 초기 크레딧 | 3,000 (보통) |
| 배경 태그 | `#추방낙인` |
| 스테이션 초기 위치 | 추방 팩션 구역 외 선택 가능 |
| 선택한 팩션 1개 호감도 | **-50** (추방한 팩션) |
| 나머지 팩션 호감도 | ±0 |

> 게임 시작 시 어느 팩션에서 추방됐는지 1개 선택 (전체 팩션 풀 중).

**`#추방낙인` 태그 효과:**
- 해당 팩션 관할 구역 진입 시 즉각 적대 반응 위험
- 해당 팩션 의뢰 접근 불가 (호감도 -50 이하 = 의뢰 차단)
- 역으로: 해당 팩션의 적 팩션에서는 환영받을 가능성 ↑

---

### 1.5 몰락자 (The Fallen)

> 있었던 지위나 자산을 잃은 자. 상층/중층 네트워크 잔재가 남아있다.

| 항목 | 값 |
|---|---|
| 초기 크레딧 | 2,000 (낮음, 몰락했으니) |
| 배경 태그 | `#몰락_귀족` |
| 스테이션 초기 위치 | 중층 / 하층 선택 가능 |
| 선택한 중층/상층 팩션 1개 호감도 | **+10** (옛 네트워크 잔재) |
| 하층 팩션 호감도 | -5 (태도가 티남) |

**`#몰락_귀족` 태그 효과:**
- 기업/상층 미션에서 예절, 의전 관련 이점 (Phase 0에서도 일부 유추 가능)
- 하층 환경에서 이질감 표적이 될 수 있음
- Infamy가 쌓일수록 "몰락"의 낙인이 옛 네트워크를 자산 → 위험으로 전환 가능

---

### 1.6 상실자 (The Lost)

> 기억, 신원, 가족, 목적 등 무언가를 잃은 자. 가장 열려있는 배경.

| 항목 | 값 |
|---|---|
| 초기 크레딧 | 3,000 (보통) |
| 배경 태그 | `#기억_공백` |
| 스테이션 초기 위치 | 전 구역 선택 가능 |
| 전 팩션 호감도 | ±0 |

**`#기억_공백` 태그 효과:**
- 이야기 이벤트 중심 배경. 특정 미션에서 과거 단서 발견 이벤트 트리거 가능
- AI 생성 시 이 태그를 활용해 "이 픽서에게 뭔가 있다"는 뉘앙스를 브리핑에 녹일 수 있음
- 시스템적 페널티/보너스는 없음 — 순수하게 이야기 레이어에서 작동

---

### 1.7 무지한 외부인 (Ignorant Outsider)

> 돔 외부의 평범한 도시·국가에서 온 자. 콜 내부 팩션·문화에 무지하지만 독립적이다.

| 항목 | 값 |
|---|---|
| 초기 크레딧 | 5,000 (많음, 외부 자산) |
| 배경 태그 | `#외부인_낙인`, `#외부_연줄` |
| 스테이션 초기 위치 | 전 구역 선택 가능 |
| 전 팩션 호감도 | ±0 (모른다) |

**`#외부인_낙인` 태그 효과:**
- 검문 시 이질적 반응. 돔 내부 신분증 체계 미등록 상태
- 하층에서 표적이 되기 쉬움 (모르는 놈 = 침범자)

**`#외부_연줄` 태그 효과:**
- 돔 외부 거래 관련 특수 미션 접근 가능 (고도화 단계)
- 외부 물자 루트 관련 지원 미션에서 특수 선택지

## 2. 스테이션 위치(Tier) × 시설 타입 가중치 테이블

### 2.1 위치(Tier) 기본 가중치

스테이션이 어느 구역에 있느냐에 따른 미션/용병 풀 기울기.

| 위치 | 하층 미션 가중치 | 중층 미션 가중치 | 상층 미션 가중치 | 운영 비용 |
|---|---|---|---|---|
| **하층** | ×1.5 | ×0.7 | ×0.2 | 낮음 |
| **중층** | ×0.8 | ×1.4 | ×0.6 | 보통 |
| **상층** | ×0.3 | ×0.8 | ×1.8 | 높음 |

### 2.2 시설 타입별 미션/용병 가중치 (기본 프로파일)

시설 선택 시 `facility_weight_profile`에 자동으로 적용.  
최종 가중치 = 위치 가중치 × 아래 시설 가중치 × 팩션 상태 가중치.

#### 숙박 (Accommodation)

| 시설 | 미션 타입 가중치 ↑ | 용병 풀 가중치 ↑ |
|---|---|---|
| 보이드 생크텀 | 잠입 ×1.5, 비밀 ×1.3 | Cypher 고스탯, `#신원세탁` 태그 |
| 에이펙스 사나토리움 | 전투 ×1.5, 지원(의료) ×1.2 | Frame 고스탯, `#전투_베테랑` 태그 |
| 마인드 팰리스 | 기업 ×1.4, 비밀 ×1.2 | Cool 고스탯, `#심리_저항` 태그 |
| 코포레이트 서버룸 | 지원(해킹) ×1.5, 비밀 ×1.2 | Wire 고스탯, 넷러너 계열 |

#### 유흥 (Entertainment)

| 시설 | 미션 타입 가중치 ↑ | 용병 풀 가중치 ↑ |
|---|---|---|
| 하이소사이어티 클럽 | 기업 ×1.5, 비밀(상층) ×1.4 | 상층 연줄 용병, `#사교_예법` 태그 |
| 메가 클럽 | 전투 ×1.3, 잠입(중하층) ×1.2 | 독립 팩션 연줄 용병, La Nada 관련 |
| 하이롤러 팰리스 | 비밀 ×1.4, 고위험 미션 비율 ↑ | Pulse 고스탯, `#도박사` 태그 |

#### 식사 (Food)

| 시설 | 미션 타입 가중치 ↑ | 용병 풀 가중치 ↑ |
|---|---|---|
| 시티 라운지 | 전투 ×1.3, 지원(크롬 집약) ×1.2 | 고임플란트 용병, `#크롬_중독` 계열 |
| 하이테크 레스토랑 | 전 타입 균등 (편향 없음) | 전 타입 균등 |
| 인텔리전스 허브 | 비밀 ×1.4, 지원(정보전) ×1.2 | Pulse 계열, `#정보원` 태그 |

#### 장비 (Equipment)

| 시설 | 미션 타입 가중치 ↑ | 용병 풀 가중치 ↑ |
|---|---|---|
| 바이오테크 클리닉 | 지원(의료) ×1.4, 비밀 ×1.2 | 바이오 임플란트 용병, `#약물_내성` |
| 쉐도우 익스체인지 | 전투 ×1.3, 잠입(암시장) ×1.3 | 하층 독립 용병, `#무기_밀수` 연줄 |
| 전술 튜닝 센터 | 잠입 ×1.4, 기업 ×1.2 | Cypher/Cool 균형 용병, `#위장_전문` |
| 정밀 공학 연구소 | 지원(기술) ×1.4, 기업 ×1.2 | Wire/Frame 균형 용병, `#기술_긱` |

#### 업무 (Business)

| 시설 | 미션 타입 가중치 ↑ | 용병 풀 가중치 ↑ |
|---|---|---|
| 섀도우 브로커 데스크 | 전 타입 접근 가능 + 리롤 권한 | 다양한 팩션 연줄 용병 (가장 넓은 풀) |
| 탤런트 매니지먼트 | 지원 ×1.3, 기업 ×1.3 (안정 의뢰 ↑) | 재계약 용병 확률 ↑, 충성도 고용병 |

## 4. 미션 타입 × 구역 기본 매핑

구역과 미션 타입 간의 자연스러운 어울림 기본값. 시설/팩션 상태로 가중치가 추가된다.

| 미션 타입 | 하층 기본 빈도 | 중층 기본 빈도 | 상층 기본 빈도 | 주요 스탯 |
|---|---|---|---|---|
| **전투** | ★★★★ | ★★★☆ | ★☆☆☆ | Frame, Pulse |
| **잠입** | ★★★☆ | ★★★★ | ★★★☆ | Cypher, Cool |
| **지원** | ★★★★ | ★★☆☆ | ★☆☆☆ | Wire, Frame |
| **기업** | ★☆☆☆ | ★★★★ | ★★★★ | Cool, Cypher |
| **비밀** | ★★☆☆ | ★★★☆ | ★★★★ | Pulse, Cool |

## 5. 의뢰인(Client) 무소속 확률

의뢰인이 특정 팩션에 소속되지 않을 확률 (익명/무소속 의뢰).

| 구역 | 무소속 확률 | 비고 |
|---|---|---|
| 하층 | 10% | 대부분 하층 팩션 의뢰 |
| 중층 | 30% | 독립 개인, 소규모 조직 |
| 상층 | 50% | 정치적 기밀 이유로 익명 선호 |
